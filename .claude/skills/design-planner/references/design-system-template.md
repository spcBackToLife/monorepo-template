## 6. 动效系统

### 6.1 缓动函数

| 名称 | 值 | 使用场景 | 视觉感受 |
|------|------|---------|---------|
| ease-default | cubic-bezier(0.4, 0, 0.2, 1) | 通用过渡 | 自然 |
| ease-in | cubic-bezier(0.4, 0, 1, 1) | 退出动画 | 加速离开 |
| ease-out | cubic-bezier(0, 0, 0.2, 1) | 进入动画 | 减速到达 |
| spring | cubic-bezier(0.175, 0.885, 0.32, 1.275) | 弹跳效果 | 活力/趣味 |

### 6.2 时长规范

| 时长 | 使用场景 | 为什么 |
|------|---------|--------|
| 100ms | 按钮press/release | 即时反馈,不能有延迟感 |
| 150ms | hover状态/微变化 | 可感知但不打断 |
| 200ms | Tab切换/简单过渡 | 平滑但不拖沓 |
| 300ms | 页面转场/Sheet弹出 | 足够看清方向 |
| 400ms | 弹跳/强调动效 | spring需要更长展示 |
| 2000-3000ms | 循环动画(呼吸/波纹) | 背景装饰不能太快 |

### 6.3 动画模式

| 模式 | 触发条件 | 效果 | 参数 |
|------|---------|------|------|
| 按压反馈 | touch/click | scale(0.96→1) | 100ms ease |
| 进入弹跳 | 元素首次出现 | scale(0→1.05→1) | 400ms spring |
| 脉冲呼吸 | 持续循环 | scale(1→1.1→1) + opacity(1→0.7→1) | 2s ease-in-out infinite |
| 波纹扩散 | 点击触发 | 从触点扩散的圆形 | 600ms ease-out |
| 渐入 | 列表项出现 | opacity(0→1) + translateY(10→0) | 200ms ease-out, stagger:50ms |
