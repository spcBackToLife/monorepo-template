# 装饰性元素设计指南

## 一、装饰元素分类体系

### 1. 几何类

| 子类型 | 形态 | 传递的感受 | 适用风格 |
|--------|------|-----------|---------|
| 直线/线段 | 平行线、交叉线、放射线 | 秩序、方向、速度 | 科技/商务/极简 |
| 网格/矩阵 | 规则网格、透视网格 | 系统、结构、技术感 | 科技/数据/后台 |
| 三角形 | 锐角、等边、组合 | 稳定(正)、动态(倒)、锐利 | 现代/潮流/体育 |
| 矩形/方块 | 像素块、色块组合 | 模块化、数字感 | 科技/游戏/开发者 |
| 多边形 | 六边形、八边形、不规则多边形 | 复杂、精密、蜂窝协作 | 科技/企业/协作 |
| 菱形/钻石 | 旋转45°方块 | 精致、premium、宝石 | 奢华/金融/高端 |

### 2. 有机类

| 子类型 | 形态 | 传递的感受 | 适用风格 |
|--------|------|-----------|---------|
| 圆/椭圆 | 实心圆、圆环、重叠圆 | 柔和、完整、和谐 | 社交/健康/儿童 |
| 光斑/散景 | 不同大小的模糊圆点 | 梦幻、温暖、空间深度 | 社交/摄影/夜景 |
| 波浪/曲线 | 正弦波、自由曲线、流体 | 流动、自然、轻松 | 音乐/艺术/生活 |
| 气泡/水滴 | 大小不一的圆润形状 | 轻盈、活泼、有趣 | 社交/通讯/儿童 |
| 云朵/烟雾 | 柔和边缘的不规则形态 | 梦幻、轻柔、抽象 | 创意/冥想/阅读 |
| 花瓣/叶片 | 有机弧线组合 | 自然、生长、生命力 | 健康/环保/美妆 |

### 3. 光效类

| 子类型 | 形态 | 传递的感受 | 适用风格 |
|--------|------|-----------|---------|
| 渐变光晕 | 大面积柔和渐变 | 空间、未来、高级感 | 科技/暗色主题/现代 |
| 光线/射线 | 从某点发散的线条 | 聚焦、重要性、能量 | 科技/游戏/庆典 |
| 镜面反射 | 高光条、玻璃反射 | 质感、真实、premium | 产品/工具/专业 |
| 霓虹/发光 | 边缘发光的形状 | 夜色、活力、电子 | 娱乐/夜生活/游戏 |
| 极光/彩虹 | 多色渐变带 | 多样性、创造力、梦想 | 创意/设计/社区 |
| 星芒/闪烁 | 十字/米字光芒 | 突出、新鲜、奖励感 | 电商/会员/成就 |

### 4. 纹理类

| 子类型 | 形态 | 传递的感受 | 适用风格 |
|--------|------|-----------|---------|
| 噪点/颗粒 | 细密的随机点 | 质感、复古、手工 | 复古/艺术/高端 |
| 纸纹/织物 | 有规律的微细纹理 | 温暖、触感、自然 | 阅读/笔记/手作 |
| 点阵/半色调 | 规则排列的圆点 | 印刷感、波普、趣味 | 潮流/艺术/复古 |
| 条纹/斜线 | 等距平行线填充 | 运动、动感、警示 | 体育/施工/标记 |

### 5. 符号类

| 子类型 | 形态 | 传递的感受 | 适用风格 |
|--------|------|-----------|---------|
| 星星 | 四角星、五角星、六角星、米字星 | 卓越、奖励、夜空、评价 | 会员/成就/评分 |
| 十字/加号 | 小十字散布 | 医疗、添加、正能量 | 医疗/效率/社区 |
| 箭头/指向 | 方向暗示 | 引导、进步、前进 | 导航/教育/成长 |
| 心形 | 各种大小心形 | 爱、喜欢、健康 | 社交/婚恋/健康 |
| 音符/波形 | 音乐符号、声波 | 韵律、音乐、声音 | 音乐/播客/通讯 |

### 6. 数学/科学类

| 子类型 | 形态 | 传递的感受 | 适用风格 |
|--------|------|-----------|---------|
| 贝塞尔曲线 | 优美的参数曲线 | 优雅、精确、设计感 | 设计/编程/工具 |
| 螺旋/涡旋 | 斐波那契/等角螺旋 | 自然规律、无限、深度 | 科学/教育/哲学 |
| 坐标系/轴线 | 笛卡尔/极坐标 | 分析、数据、精确 | 数据/金融/研究 |
| 分子/连接 | 节点+连线 | 网络、关系、科技 | 社交/AI/区块链 |
| 轨道/环绕 | 椭圆轨道路径 | 系统、宇宙、周期 | 科技/太空/系统 |

### 7. 裁剪/溢出装饰类（Clipped Shapes）

这是一类非常重要且常见的装饰手法：**图形只露出一部分，被容器边缘裁剪**。

| 子类型 | 形态 | 传递的感受 | 适用风格 |
|--------|------|-----------|---------|
| 截断矩形/色块 | 矩形伸出画面边缘，只露出一角或一边 | 空间延伸、现代、层次 | 企业/SaaS/现代 |
| 半圆/四分之一圆 | 圆形被裁剪，只露出弧线部分 | 柔和、框架感、包围 | 社交/教育/金融 |
| 斜切色块 | 用斜线/弧线切割的大面积色块 | 动态、区分、引导 | 营销/落地页/产品 |
| 溢出圆环 | 大圆环从角落伸出，大部分在画面外 | 氛围、深度、宏大 | 科技/暗色/仪表盘 |
| 弧线分区 | 用曲线替代直线来分隔页面区域 | 流动、自然、不呆板 | 生活/社交/内容 |
| 不规则色块(Blob) | 自由形状的大面积渐变填充 | 有机、亲和、现代 | 科技/SaaS/社交 |
| 叠层卡片 | 多个矩形/圆角矩形错位重叠 | 层次、丰富、信息感 | 产品/展示/作品集 |

#### 裁剪装饰的设计思考逻辑

```
为什么截断比完整更好？

1. 完整的圆/矩形 = 闭合 = 视觉终止 = 显得狭小
   截断的圆/矩形 = 开放 = 暗示画面外延伸 = 空间感

2. 完整图形占据确定面积 = 跟内容"抢地盘"
   截断图形面积不确定 = 在背景层"若隐若现" = 不干扰内容

3. 完整图形有明确边界 = 生硬
   截断图形边界被裁掉 = 柔和融入页面 = 自然
```

#### 常见组合模式

```
模式 A: 对角色块
  左上角: 大半圆(primary 10%) 从左上角溢出
  右下角: 小截断矩形(secondary 8%) 从右下角伸出
  效果: 对角线方向引导 + 空间感

模式 B: 弧线分区
  页面上部: 主色区域，底部用弧线/波浪结束
  页面下部: 背景色区域
  效果: 自然分区，比直线分割更优雅

模式 C: 多层叠色块
  底层: 大面积浅色块(占 60% 画面)
  中层: 中等色块(偏移重叠, 30% 画面)
  顶层: 内容卡片
  效果: 深度感 + 区域划分 + 丰富视觉

模式 D: 角落光圈
  4 个角落各有一个大圆(只露 1/4)
  透明度递减: 近角 15% → 远角 5%
  效果: 柔和的框架感，像聚光灯打在中间内容上
```

#### 色彩过渡手法

```
相邻色块之间如何过渡：

1. 硬切 — 两个色块直接相邻，用 contrast 制造层次
   适用: 几何/极简/强对比风格

2. 渐变融合 — 两个色块之间用 gradient 过渡
   适用: 柔和/现代/有机风格

3. 模糊边缘 — 色块边缘使用大幅 blur 柔化
   适用: 梦幻/科技/暗色风格

4. 重叠透明 — 两个半透明色块重叠区域产生混色
   适用: 创意/艺术/玻璃态风格

5. 噪点过渡 — 边缘用抖动/噪点打散，避免生硬
   适用: 复古/质感/手工风格
```

## 二、选择逻辑

### 2.1 从品牌调性推导装饰风格

```
品牌调性 → 主装饰类型 + 辅助装饰类型

科技/未来    → 主: 几何(网格/线段) + 光效(光晕/射线)    辅: 数学(曲线)
社交/温暖    → 主: 有机(光斑/气泡) + 符号(心形/星星)    辅: 光效(渐变)
商务/专业    → 主: 几何(直线/矩形) + 纹理(噪点)         辅: 光效(镜面)
创意/艺术    → 主: 有机(波浪/花瓣) + 纹理(半色调)       辅: 光效(极光)
游戏/活力    → 主: 符号(星星/闪烁) + 几何(碎片)         辅: 光效(霓虹)
极简/高端    → 主: 几何(细线) + 光效(微妙渐变)          辅: 纹理(噪点)
教育/成长    → 主: 符号(星星/箭头) + 有机(圆/光斑)      辅: 几何(点阵)
```

### 2.2 从页面功能推导装饰用量

```
页面功能              装饰密度      原因
─────────────────────────────────────────────
登录/注册             少量(2-3)     焦点在表单，装饰只是氛围
首页/Feed             极少(1-2)     内容本身就是"装饰"，额外装饰会干扰
空状态/错误           中等(5-8)     需要填充空白 + 引导情绪
引导页/Onboarding     中等(5-8)     需要营造品牌第一印象
个人中心/Profile      少量(2-4)     强调个人内容，装饰辅助个性化
设置/列表类           极少(0-1)     纯功能页，装饰=噪音
庆典/成就/会员        密集(8-15)    需要营造兴奋/尊贵感
```

### 2.3 从视觉层次推导装饰位置

```
装饰只能出现在 L4 氛围层（最低视觉优先级）：

┌─────────────────────────────┐
│  L1 前景交互层               │ ← 绝对不放装饰（干扰操作）
│  ─────────────────────────  │
│  L2 内容层                   │ ← 不放装饰（干扰阅读）
│  ─────────────────────────  │
│  L3 结构层                   │ ← 极克制的装饰（分割线可带微渐变）
│  ─────────────────────────  │
│  L4 氛围层 ← 装饰在这里     │ ← 背景/角落/边缘/底层
└─────────────────────────────┘

具体位置策略：
  角落: 光晕/几何图案放在四角（不占中心注意力）
  边缘: 波浪/曲线沿顶部或底部（空间分隔+氛围）
  背景: 大面积低对比度渐变/纹理（整体调性）
  点缀: 小符号(星/点/十字)散布在空白区域（活力感）
```

## 三、组合规则

### 3.1 同类不超过 2 种变体

```
❌ 同时出现: 圆形光斑 + 椭圆光斑 + 环形 + 模糊圆（太杂乱）
✅ 最多 2 种: 圆形光斑(大) + 环形(小) — 有主有次
```

### 3.2 跨类最多 3 种类型组合

```
❌ 几何线条 + 光斑 + 星星 + 波浪 + 噪点 （5种 = 视觉灾难）
✅ 几何线条 + 渐变光晕 + 微妙星星点缀 （3种 = 丰富但不杂乱）
```

### 3.3 大小/透明度必须有层次

```
装饰元素的视觉重量梯度（从强到弱）：
  1个主装饰: 大面积 + 15-25% 透明度（定调性）
  2-3个次装饰: 中等面积 + 5-15% 透明度（丰富感）
  N个微装饰: 小面积 + 3-8% 透明度（精致感）

❌ 所有装饰一样大一样亮 → 没有层次 = 噪音
✅ 主次分明 → 有节奏有呼吸
```

### 3.4 颜色控制

```
装饰颜色规则：
  ├── 同色系: 全部使用 primary 的不同透明度（5%/10%/20%）— 最安全
  ├── 双色系: primary + secondary 交替 — 丰富但和谐
  └── 多色系: 仅用于庆典/游戏/创意场景 — 需要极强的色彩控制力

❌ 装饰颜色比内容还鲜艳 → 喧宾夺主
✅ 装饰颜色永远比内容淡（低饱和/低透明度）
```

## 四、用量控制的决策树

```
Q1: 这个页面的核心内容是什么？
  ├── 内容本身很丰富（Feed/图片/视频）→ 装饰极少(0-2)
  └── 内容简单/空白较多（登录/空态/设置）→ 继续 Q2

Q2: 页面的情感需求是什么？
  ├── 纯功能（设置/列表）→ 几乎不要装饰(0-1)
  ├── 品牌表达（登录/注册/启动）→ 少量精致装饰(2-4)
  ├── 情绪引导（空态/错误/成就）→ 中等装饰(4-8)
  └── 庆典/特殊（会员/活动/节日）→ 可以密集(8-15)

Q3: 装饰的主次关系？
  ├── 1 个主视觉(占 60% 视觉重量) + N 个点缀(占 40%)
  └── 禁止：所有装饰同等重要 = 没有重点

Q4: 去掉任何一个装饰后，页面变差了吗？
  ├── 是 → 保留
  └── 否 → 删除（"如果不知道该不该加，就不加"）
```

## 五、不同风格的装饰配方

### 科技暗色风（如 CampusLink 当前风格）

```
主装饰: 渐变光晕(primary 15%, 右上角, 大面积模糊)
次装饰: 细几何线条(primary 8%, 对角方向, 2-3条)
微装饰: 小光点(3-5个, 随机散布, 2-4px, primary 10%)
配色: 仅使用 primary 色的不同透明度
总数: 6-8 个元素
```

### 清新浅色风（社交/生活类）

```
主装饰: 大圆形渐变色块(secondary 10%, 偏左下)
次装饰: 波浪曲线(primary 6%, 底部)
微装饰: 小气泡/圆点(4-6个, primary/secondary 5%)
配色: primary + secondary 双色
总数: 7-10 个元素
```

### 极简高端风（工具/专业类）

```
主装饰: 无（留白即是设计）
次装饰: 一条极细线(border-color 3%, 水平分隔)
微装饰: 微妙噪点纹理(全局 2% 覆盖)
配色: 仅黑白灰 + 极少量 primary 点缀
总数: 1-3 个元素
```

### 活力游戏风（年轻/娱乐类）

```
主装饰: 不规则几何碎片(多色, 15%, 动态旋转)
次装饰: 星芒/闪烁(primary/secondary, 随机位置)
微装饰: 小十字/圆点阵列(4-6px, 10%, 重复图案)
配色: 多色(primary + secondary + accent)
总数: 10-15 个元素
```

## 六、装饰与用户体验的关系

```
装饰的终极目的不是"好看"，而是：

1. 引导注意力    → 光晕指向 CTA 按钮方向
2. 建立空间感    → 前景清晰 + 背景模糊 = 深度
3. 传递品牌调性  → 看到装饰就知道是什么品牌/风格
4. 填充空白      → 避免页面"空荡荡"但不能用装饰填满
5. 营造情绪      → 温暖/冷静/兴奋/专注
6. 暗示可交互    → 按钮光效暗示"可以点"

每个装饰都必须至少服务于上述 1 个目的。
无目的的装饰 = 视觉噪音 = 必须删除。
```
