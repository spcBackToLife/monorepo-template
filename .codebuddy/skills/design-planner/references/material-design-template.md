# 素材独立文档模板

每个素材是一个独立 `.md` 文件，放在所属页面或组件的 `materials/` 目录下。

**文件命名**: `[ID]-[name].md`，如 `I-01-locate-icon.md`、`D-02-glow-circle.md`

**ID 前缀规则**:
- `I-xx`: Icon（功能图标）
- `D-xx`: Decoration（装饰元素）
- `B-xx`: Brand（品牌素材）
- `IL-xx`: Illustration（插画）

**存放位置**:
```
pages/[NN]-[name]/materials/[ID]-[name].md    ← 页面级素材 & 页面组件内素材
components/[NN]-[name]/materials/[ID]-[name].md  ← 通用组件内素材
```

本模板定义了每个素材文档必须包含的完整 6 节结构，不可缺少。
