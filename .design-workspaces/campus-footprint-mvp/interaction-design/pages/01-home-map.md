# 足迹地图（主页）— 交互规格

---

## 状态机

```
States:
  loading         → 首次进入，加载地图+附近动态
  idle            → 地图已加载，展示附近动态气泡
  browsing        → 用户正在地图上浏览/缩放/平移
  bubble_preview  → 点击了某个气泡，展示简要预览卡片
  detail          → 进入动态详情页
  no_location     → 位置权限未授权
  out_of_campus   → 在校园围栏外
  offline         → 无网络
  empty           → 附近无动态

Transitions:
  loading → idle:            地图+动态数据加载完毕
  loading → no_location:     位置权限未授权
  loading → offline:         无网络
  loading → out_of_campus:   GPS在校园围栏外
  idle → browsing:           用户开始拖动/缩放地图
  idle → bubble_preview:     点击某个动态气泡
  idle → empty:              附近10米内无任何动态
  browsing → idle:           停止操作1.5秒后自动归位
  bubble_preview → detail:   点击预览卡片「查看详情」
  bubble_preview → idle:     点击空白区域关闭预览
  detail → idle:             返回地图
  no_location → idle:        用户授权后重新定位成功
  out_of_campus → idle:      用户进入校园范围
  offline → loading:         网络恢复后自动重试

Effects:
  → loading:        地图骨架屏（灰色地图轮廓+脉冲动画）
  → idle:           气泡从透明→实体依次出现（由近到远，50ms间隔）
  → bubble_preview: 气泡放大1.2x + 底部弹出预览卡片(Bottom Sheet 30%高)
  → no_location:    全屏引导「开启位置，发现身边的足迹」+ CTA去设置
  → out_of_campus:  顶部横幅「你当前不在校园范围内，部分功能受限」
  → empty:          地图中心显示轻插画「这里还没有人留下足迹，成为第一个吧 ✨」
  → offline:        顶部红色横幅「网络已断开」+ 显示缓存的旧数据
```

---

## 核心交互详解

### 1. 地图浏览

| 操作 | 效果 |
|------|------|
| 自动定位 | 进入后以用户位置为中心，半径100米视野 |
| 双指缩放 | 平滑缩放，最小50米视野/最大500米 |
| 单指拖拽 | 平移地图，松手后不自动回位 |
| 点击「回到我」按钮 | 平滑动画回到用户当前位置居中 |
| 地图旋转 | 禁用（始终北朝上，降低认知负荷） |

### 2. 动态气泡展示

**气泡样式规则**：
| 类型 | 样式 | 动效 |
|------|------|------|
| 公开动态（未读） | 蓝色圆形气泡 + 白色数字 | 微弱浮动（上下2px循环） |
| 公开动态（已读） | 灰色半透明气泡 | 静止 |
| 定向给我的（未读） | 金色气泡 + ✨闪烁 | 脉冲放大（1.0→1.1循环） |
| 定向给我的（已读） | 金色半透明 | 静止 |
| 多条动态聚合 | 气泡内数字 | 如「3」表示该位置3条 |

**气泡出现逻辑**：
- 只显示 10 米内的动态（核心产品规则）
- 用户移动时实时刷新（500ms节流）
- 新动态出现：从地面「冒出」+ 弹跳动画

### 3. 预览卡片交互

点击气泡后，底部弹出 30% 高度的预览卡片：

```
┌────────────────────────────────────┐
│ ─── (拖拽条)                       │
│                                    │
│ [头像] 用户名        2分钟前        │
│                                    │
│ 「今天在图书馆学到崩溃😭…」          │
│ [缩略图] [缩略图]                  │
│                                    │
│ ❤️ 12   💬 3          [查看详情 →]  │
└────────────────────────────────────┘
```

- 上滑可展开到 60%（显示更多内容）
- 下滑关闭
- 点击「查看详情」→ Push 到动态详情页
- 如果该位置有多条动态：卡片内可左右滑动切换

### 4. 位置触发体验

用户走到有动态的位置时（后台地理围栏触发）：

```
步骤：
1. 系统推送：「附近有 N 条足迹等你发现 👣」
2. 点击推送 → 打开App → 自动定位到当前位置
3. 顶部出现金色横幅（App内）：「你发现了 3 条新足迹！」
4. 新发现的气泡有高亮入场动画（从地面弹出+光圈）
5. 横幅3秒后自动收起（可点击横幅快速查看列表）
```

### 5. 视图切换

- 右上角切换按钮：地图视图 ⟷ 列表视图
- 切换动画：交叉淡入淡出 200ms
- 列表视图：按距离排序，每条显示「距离你 X 米」
- 列表中的「距离」实时更新（用户移动时）

## 操作清单

| 操作 | 触发 | 即时反馈(L0) | 进行中 | 成功 | 失败 |
|------|------|------------|--------|------|------|
| 点击气泡 | tap | 气泡scale(1.2) | — | 底部弹出预览卡片 | — |
| 点赞(预览卡) | tap ❤️ | 心跳放大+变红 | — | 数字+1 | Toast错误 |
| 下拉刷新 | pull | 顶部指示器 | 旋转 | 气泡重新加载 | Toast |
| 回到我 | tap按钮 | scale(0.96) | 地图平滑移动 | 居中完成 | — |
| 切换视图 | tap按钮 | 图标旋转 | — | 淡入新视图 | — |
| 点击发布 | tap「+」 | scale+弹跳 | — | Push发布页 | — |

## 加载策略

| 场景 | 策略 |
|------|------|
| 首次进入 | 地图骨架屏（1-2秒）→ 地图渲染 → 气泡依次出现 |
| 移动后更新 | 静默刷新（无UI变化），新气泡渐入 |
| 下拉刷新 | 顶部spinner + 保留当前气泡 → 更新完替换 |
| 切换到列表 | 如已有数据直接展示，否则骨架屏 |

## 边界情况

| 情况 | 处理 |
|------|------|
| GPS漂移导致气泡闪烁 | 位置变化<5米忽略，不重新计算可见范围 |
| 100+条动态聚集 | 缩放级别低时聚合为一个大气泡+数字 |
| 用户快速移动（骑车/公交） | 降低刷新频率到2秒/次 |
| 后台切回前台 | 静默刷新位置+动态 |
| 首次无任何动态 | 空状态+「发布第一条」CTA+附近热门位置推荐 |
