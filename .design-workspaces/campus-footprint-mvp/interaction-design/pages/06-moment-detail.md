# 动态详情页 — 交互规格

---

## 状态机

```
States:
  loading       → 加载动态详情
  loaded        → 内容展示
  commenting    → 正在输入评论
  submitting    → 评论/点赞提交中
  deleted       → 动态已被删除
  error         → 加载失败

Transitions:
  loading → loaded:       数据返回
  loading → error:        网络错误
  loading → deleted:      动态已不存在
  loaded → commenting:    点击评论输入框
  commenting → submitting: 发送评论
  submitting → loaded:    评论成功（追加到列表）
  submitting → commenting: 评论失败（内容保留）

Effects:
  → loading:     骨架屏（头像圆+文字条+图片占位）
  → loaded:      内容淡入，评论列表从下往上依次出现
  → commenting:  底部输入框上移+键盘弹出+页面上推
  → deleted:     全屏「该动态已被删除」+ 返回按钮
```

## 核心交互

### 点赞
- tap ❤️ → 心跳放大动画(scale 1→1.3→1) + 变红 + 数字+1
- 已赞再tap → 取消：缩小+变灰 + 数字-1
- 乐观更新：UI即时变化，失败后回滚+Toast

### 评论
- 点击输入框 → 聚焦+键盘弹出
- 发送：按钮loading → 评论插入列表顶部（带入场动画）
- 长按评论：弹出「复制/举报」菜单（自己的评论：「复制/删除」）

### 图片浏览
- 点击图片 → 全屏图片浏览器（黑色背景+左右滑动）
- 双指缩放+双击放大
- 下拉关闭（缩小回原位的shared element动画）

### 右上角「...」菜单
- 自己的动态：「删除」（确认弹窗）
- 别人的动态：「举报」/「屏蔽该用户」

---

## 边界情况

| 情况 | 处理 |
|------|------|
| 评论过长 | 展开/收起（超3行折叠） |
| 评论数>50 | 分页加载，「查看更多评论」 |
| 作者已注销 | 显示「已注销用户」灰色头像 |
| 图片加载慢 | 渐进式加载（模糊→清晰） |
