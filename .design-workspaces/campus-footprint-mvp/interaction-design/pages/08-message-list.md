# 消息列表 — 交互规格

---

## 状态机

```
States:
  loading       → 首次加载
  loaded        → 列表展示
  empty         → 无任何消息
  refreshing    → 下拉刷新中

Transitions:
  loading → loaded:     数据返回
  loading → empty:      无数据
  loaded → refreshing:  下拉
  refreshing → loaded:  刷新完成
```

## 核心交互

### Tab 切换
- **私信**：对话列表（按最后消息时间排序）
- **互动**：点赞/评论/被捞 聚合通知
- **系统**：官方公告/活动通知
- 切换无网络请求（本地数据切换），fade 200ms

### 私信列表
- 未读对话：加粗+右侧红色数字角标
- 左滑操作：[置顶] [删除]
- 点击进入聊天页（Push）

### 互动列表
- 格式：「[头像] 小红 赞了你的动态 [缩略图]」
- 聚合：「小红等3人赞了你的动态」
- 点击→跳转到对应动态详情

### 系统通知
- 格式：「[系统图标] 标题 \n 正文摘要 时间」
- 位置触发通知特殊样式：金色边框+📍图标

### 全局未读角标
- 底部Tab「消息」上显示红点（有未读）或数字（≤99+）

## 边界情况

| 情况 | 处理 |
|------|------|
| 新消息到达 | 实时更新列表+排序置顶+红点 |
| 删除对话 | 左滑+确认 → 列表动画移除 |
| 全部已读 | 右上角「全部已读」→ 所有红点/数字消失 |
| 被屏蔽者发消息 | 不显示（静默丢弃） |
