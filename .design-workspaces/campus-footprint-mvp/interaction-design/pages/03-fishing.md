# 捞人主页 — 交互规格

---

## 状态机

```
States:
  idle            → 主页面，显示可用次数和撒网按钮
  casting         → 撒网动效进行中（2-3秒）
  result          → 展示捞到的用户卡片
  empty_result    → 捞到0人（附近无人）
  greeting        → 正在发送打招呼
  exhausted       → 今日免费次数用完
  purchasing      → 购买道具中
  no_location     → 位置权限未授权
  out_of_campus   → 不在校园范围
  onboarding      → 首次进入引导态

Transitions:
  idle → onboarding:       首次进入捞人Tab
  onboarding → idle:       引导完成
  idle → casting:          点击「撒网」按钮（有次数）
  idle → exhausted:        次数=0时点击撒网
  exhausted → purchasing:  点击「获取更多」
  purchasing → idle:       购买成功（次数+1）
  casting → result:        匹配到1-3人
  casting → empty_result:  匹配到0人
  result → idle:           所有卡片操作完/点击完成
  result → greeting:       点击「打招呼」
  greeting → result:       打招呼发送完毕
  empty_result → idle:     点击返回
  idle → no_location:      检测到未授权位置
  idle → out_of_campus:    不在校园范围内

Effects:
  → idle:           水面背景+微波纹+底部撒网按钮+剩余次数
  → casting:        网甩出动效(500ms) → 水面冒泡(1.5-2s) → 卡片浮出
  → result:         卡片从水面浮出+水花(每张间隔200ms)
  → empty_result:   空网收回动效 + 安慰文案浮现
  → exhausted:      按钮变灰 + 弹出「次数用完」提示
  → greeting:       小信封飞出动效
  → onboarding:     3步引导浮层（半透明遮罩+高亮指引）
```

## 核心交互详解

### 1. 撒网按钮

**idle态**：
- 大圆形按钮（居中），水面涟漪动画从按钮向外扩散
- 按钮文字：「撒网🎣」
- 按钮下方：次数点●（实心=可用，空心=已用）

**点击后**：
- L0 即时：按钮scale(0.9)+Heavy震动
- 按钮变为「收网中...」+ 水面动效加剧
- 网从按钮位置甩向水面（Lottie动画，500ms）
- 水面冒泡等待（1.5-2秒，模拟搜索）
- 结果浮出 或 空网收回

**禁用态（次数=0）**：
- 按钮灰色 + 文字「已用完」
- 点击时弹出提示Sheet：
  ```
  今日免费次数已用完
  [购买额外撒网 ¥1] [使用道具]
  明天00:00刷新免费次数
  ```

### 2. 模式切换（底部3Tab）

| 模式 | 按钮态 | 点击效果 |
|------|--------|---------|
| 基础撒网 | 默认选中 | 正常撒网流程 |
| 炸弹💣 | 有库存→亮色，无库存→灰色+数字0 | 有→切换模式，无→跳转商店 |
| 精准🎯 | 同上 | 有→展开筛选条件面板 |

切换时中央按钮和视觉跟随变化：
- 炸弹模式：按钮变为「投掷💣」，水面颜色偏红/暖
- 精准模式：按钮变为「撒精准网」，水面颜色偏紫/冷

### 3. 结果卡片展示

捞到用户后，卡片从水面浮出：

```
┌─────────────────────────────────┐
│  ┌────┐                         │
│  │头像│  小明同学                │
│  └────┘  「篮球搭子在哪里！」    │
│                                  │
│  🎓 计算机 · 大三                │
│  🏷️ 篮球 编程 电影               │
│  📍 约120米                      │
│                                  │
│  [👋 打招呼]      [→ 感兴趣]     │
│                 [跳过 ↓]         │
└─────────────────────────────────┘
```

**卡片操作**：
- 左滑 = 跳过（卡片滑出+下一张浮入）
- 右滑 = 感兴趣（卡片飞出+❤️动效）
- 点击「打招呼」→ 弹出打招呼Sheet
- 所有卡片操作完 → 总结页：「本次捞到3人，打了2个招呼」

### 4. 打招呼交互

```
┌─────────────────────────────────┐
│ ─── (拖拽条)                     │
│                                  │
│  向 小明 打招呼                   │
│                                  │
│  快捷模板：                      │
│  [Hi~ 很高兴认识你]              │
│  [同校的缘分~]                   │
│  [约一局游戏？🎮]                │
│  [一起去食堂吗？🍜]              │
│                                  │
│  或自定义：                      │
│  ┌──────────────────────────┐   │
│  │ 输入自定义消息...        │   │
│  └──────────────────────────┘   │
│                                  │
│  [发送打招呼]                    │
└─────────────────────────────────┘
```

- 选中模板后高亮+按钮激活
- 自定义输入后按钮激活
- 发送：信封飞出动效 + 震动Success + Sheet收起 + Toast「打招呼已发送」
- 失败：Toast「发送失败，请重试」+ Sheet保持

### 5. 炸弹玩法

- 点击「投掷💣」→ 炸弹落入水面动效（有力度的splash）
- 爆炸！→ 震屏 + Heavy×2震动 + 大量水花
- 8-15张卡片从四面八方飞入 → 形成网格/瀑布墙
- 用户逐张浏览（可翻页或滑动）
- 操作同基础卡片（打招呼/感兴趣/跳过）

### 6. 精准网玩法

- 选择精准模式 → 展开筛选条件面板
- 设定后点击「撒精准网」→ 动效启动
- **第1次**：出现模糊卡片（高斯模糊强 + 仅显示1个标签）
  - 底部：「再拉一次看得更清楚... (2/3)」+ 30秒倒计时
- **第2次**（30秒后可用）：模糊减弱 + 头像轮廓可见 + 2个标签
  - 底部：「最后一次！(3/3)」+ 30秒倒计时
- **第3次**：完全清晰！卡片完整展示
  - 欢呼动效 + 「找到了！」

倒计时期间按钮灰色禁用 + 倒计时数字动画。

## 操作清单

| 操作 | 触发 | 即时反馈 | 成功 | 失败 |
|------|------|---------|------|------|
| 撒网 | tap | scale+震+网甩出 | 卡片浮出 | 空网收回 |
| 投炸弹 | tap | scale+震+炸弹落入 | 卡片雨 | Toast |
| 撒精准网 | tap | scale+震+网甩出 | 模糊卡片出现 | Toast |
| 打招呼 | tap | 信封飞出+震 | Toast成功 | Toast失败 |
| 左滑跳过 | swipe | 卡片滑出 | 下一张浮入 | — |
| 右滑感兴趣 | swipe | 卡片飞+❤️ | 下一张浮入 | — |
| 购买道具 | tap | loading | 次数+1+Toast | 支付弹窗/失败 |
| 切换模式 | tap | Tab高亮切换 | 按钮/背景变 | — |

## 加载策略

| 场景 | 策略 |
|------|------|
| 进入页面 | 立即显示（数据量小，次数+库存从缓存读） |
| 撒网匹配 | 定制动效替代loading（水面冒泡2秒） |
| 用户卡片头像 | 渐进加载（先色块→图片） |
| 商店页 | Push进入，骨架屏 |

## 边界情况

| 情况 | 处理 |
|------|------|
| 快速重复点击撒网 | 第一次点击后立即禁用，动效期间不可再点 |
| 匹配中网络断开 | 动效结束后Toast「网络异常」+次数不扣 |
| 捞到自己认识的人 | 正常展示（熟人也可以打招呼） |
| 所有卡片都跳过 | 结束页：「都不感兴趣？试试精准网吧 🎯」|
| 打招呼达到每日上限(10次) | 提示「今日打招呼次数已用完，明天再来」|
| 精准网没有符合条件的人 | 第3次仍为空→「没有找到匹配的人，试试放宽条件」|
