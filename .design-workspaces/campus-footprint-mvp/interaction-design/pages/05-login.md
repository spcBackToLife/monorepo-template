# 登录页 — 交互规格

---

## 状态机

```
States:
  idle            → 初始态，表单空白
  inputting       → 用户正在输入
  submitting      → 登录请求中
  success         → 登录成功
  error           → 登录失败
  locked          → 多次失败账户临时锁定
  code_login      → 验证码快捷登录模式

Transitions:
  idle → inputting:         聚焦任意输入框
  inputting → submitting:   点击登录（手机号+密码非空）
  submitting → success:     API 200
  submitting → error:       401（密码错误）/ 404（账号不存在）/ 网络错误
  error → inputting:        修改任意输入
  error → locked:           连续5次失败
  locked → idle:            锁定时间到期（自动刷新）
  idle → code_login:        点击「验证码登录」
  code_login → idle:        点击「密码登录」

Effects:
  → submitting:   按钮spinner + 所有输入禁用
  → success:      按钮变✓绿色 + 0.5s跳转主页
  → error:        按钮恢复 + 表单shake + 错误文字淡入 + 密码框清空聚焦
  → locked:       全表单禁用 + 显示倒计时「请N分钟后再试」
```

## 操作清单

| 操作 | 触发 | 即时反馈 | 成功 | 失败 |
|------|------|---------|------|------|
| 登录 | tap | scale+spinner | ✓+跳转 | shake+红字+清密码 |
| 验证码登录 | tap | 切换表单（fade） | 同密码登录 | 同密码登录 |
| 忘记密码 | tap | Push到重置页 | — | — |
| 微信登录 | tap | 跳转微信 | 回调成功→主页 | Toast失败 |
| 注册 | tap | Push到注册页 | — | — |

## 错误处理

| 错误 | UI 表现 |
|------|---------|
| 手机号格式错误 | inline：「请输入正确的手机号」 |
| 账号不存在 | inline：「该账号未注册」+「去注册→」 |
| 密码错误 | inline：「密码错误，还可尝试N次」 |
| 网络错误 | Toast：「网络异常，请检查网络连接」 |
| 账户锁定 | 全表单禁用+红色文案+倒计时 |

---

## 边界情况

| 情况 | 处理 |
|------|------|
| 新设备首次登录 | 成功后弹出短信二次验证 |
| 记住手机号 | 登录成功后缓存手机号，下次自动填充 |
| 键盘遮挡按钮 | 页面自动上移，确保按钮可见 |
| 横屏 | 不支持横屏，锁定竖屏 |
