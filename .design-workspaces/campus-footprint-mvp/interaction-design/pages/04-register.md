# 注册流程 — 交互规格

---

## 状态机

```
States:
  welcome         → 欢迎/引导页
  phone_input     → 输入手机号
  code_sending    → 验证码发送中
  code_input      → 输入验证码
  password_set    → 设置密码
  profile_setup   → 填写基本资料
  location_perm   → 位置权限引导
  complete        → 注册完成

Transitions:
  welcome → phone_input:        点击「开始」
  phone_input → code_sending:   输入11位手机号 + 点击「获取验证码」
  code_sending → code_input:    验证码发送成功（~1秒）
  code_sending → phone_input:   发送失败（提示重试）
  code_input → password_set:    6位验证码验证通过
  code_input → code_input:      验证码错误（提示重试）
  password_set → profile_setup: 密码设置完成
  profile_setup → location_perm: 资料提交完成
  location_perm → complete:     授权位置/跳过
  complete → (主页):             1.5秒后自动跳转

Effects:
  → welcome:        左右滑动引导图（3页）+ 底部「开始」按钮
  → phone_input:    手机号输入框自动聚焦 + 键盘弹起
  → code_sending:   获取按钮→spinner + 60秒倒计时开始
  → code_input:     6个方格输入框 + 自动聚焦第1格
  → password_set:   密码输入框 + 强度指示条
  → profile_setup:  昵称/性别/学校/兴趣 分步展示
  → location_perm:  全屏插图 + 说明文案 + 「开启位置」按钮
  → complete:       ✓ 大图标 + 「欢迎加入！」+ 撒花动效
```

## 分步页面详解

### Step 1: 欢迎页（3张引导图）

```
[引导图1] 📍 留下你的足迹
  「在校园的任何角落，留下只有走到这里才能看到的动态」

[引导图2] 🎣 捞到有趣的人
  「像撒网捞鱼一样，认识你身边的同学」

[引导图3] 💭 多年后的回忆
  「毕业后回到校园，看到你曾留下的一切」

───────────────────
○ ○ ●           [开始]
```

- 左右滑动切换，底部点指示器
- 最后一张「开始」按钮渐入
- 非首次安装用户跳过直接到登录

### Step 2: 手机号输入

```
┌─────────────────────────────────┐
│ ← 返回                          │
│                                  │
│  输入手机号                      │
│  我们将发送验证码确认你的身份     │
│                                  │
│  +86 ┌───────────────────────┐  │
│      │ 请输入手机号           │  │
│      └───────────────────────┘  │
│                                  │
│  [获取验证码]  ← 灰色禁用       │
│                                  │
│  已有账号？去登录                │
└─────────────────────────────────┘
```

**交互**：
- 进入自动聚焦，弹出数字键盘
- 实时格式化：138 0000 0000（3-4-4）
- 输入11位后按钮自动激活（亮色）
- 格式错误（非1开头/不足11位）点击按钮：inline红字「请输入正确的手机号」
- 已注册：提示「该手机号已注册」+ 「去登录→」链接

### Step 3: 验证码输入

```
┌─────────────────────────────────┐
│ ← 返回                          │
│                                  │
│  输入验证码                      │
│  已发送到 138****0000           │
│                                  │
│  ┌─┐ ┌─┐ ┌─┐ ┌─┐ ┌─┐ ┌─┐    │
│  │ │ │ │ │ │ │ │ │ │ │ │    │
│  └─┘ └─┘ └─┘ └─┘ └─┘ └─┘    │
│                                  │
│  重新获取 (52s)                  │
│                                  │
└─────────────────────────────────┘
```

**交互**：
- 6个独立方格，自动聚焦第1格
- 输入一位自动跳到下一格（带光标移动动画）
- 删除：回退到上一格
- 粘贴6位数字：自动填满所有格
- 填满6位→自动提交验证（不需要点按钮）
- 验证中：所有格加载态（边框脉冲）
- 验证失败：所有格变红+shake+清空+重新聚焦第1格
- 倒计时60秒，归0后「重新获取」可点击
- 3次错误后：强制等待60秒重新获取

### Step 4: 设置密码

```
┌─────────────────────────────────┐
│ ← 返回                          │
│                                  │
│  设置密码                        │
│  6-20位，需包含字母和数字        │
│                                  │
│  ┌─────────────────────────┐    │
│  │ 请设置密码          [👁️] │    │
│  └─────────────────────────┘    │
│  ▪▪▪▪▪░░░░░░░░░░ (弱/中/强)    │
│                                  │
│  ┌─────────────────────────┐    │
│  │ 确认密码            [👁️] │    │
│  └─────────────────────────┘    │
│                                  │
│  [下一步]                        │
└─────────────────────────────────┘
```

**交互**：
- 默认密码不可见（•••），点击眼睛切换
- 强度指示条实时更新：
  - 红色(弱)：纯数字/纯字母/长度<8
  - 橙色(中)：字母+数字，8-12位
  - 绿色(强)：字母+数字+特殊字符/长度>12
- 两次密码不一致：失焦时第二个框下方红字「两次密码不一致」
- 都OK后「下一步」激活

### Step 5: 基本资料

```
┌─────────────────────────────────┐
│ ← 返回              跳过 →      │
│                                  │
│  完善你的资料                    │
│  让别人更好地认识你              │
│                                  │
│  昵称*                           │
│  ┌───────────────────────────┐  │
│  │ 给自己取个名字             │  │
│  └───────────────────────────┘  │
│                                  │
│  性别*     ○ 男    ○ 女          │
│                                  │
│  学校*     [选择学校 ▶]          │
│                                  │
│  年级*     [选择年级 ▶]          │
│                                  │
│  兴趣标签 (选填，最多5个)        │
│  [篮球] [摄影] [游戏] [编程]..  │
│                                  │
│  [完成注册]                      │
└─────────────────────────────────┘
```

**交互**：
- 标*为必填，其余选填
- 学校选择：Push到学校搜索页（拼音搜索+热门推荐）
- 年级：底部滚轮选择器
- 兴趣标签：气泡式多选，选中弹跳+变色，最多5个
- 选满5个后其余标签变灰不可选
- 「跳过」：跳过兴趣标签（昵称/性别/学校必填不可跳）
- 完成：按钮loading → 成功

### Step 6: 位置权限

```
┌─────────────────────────────────┐
│                                  │
│        [位置插图]                │
│                                  │
│   开启位置，发现身边的足迹 📍    │
│                                  │
│   • 看到附近同学留下的动态       │
│   • 捞人认识身边的人             │
│   • 收到TA给你的专属留言通知     │
│                                  │
│   [开启位置服务]  ← 主按钮       │
│                                  │
│   稍后再说                       │
│                                  │
└─────────────────────────────────┘
```

- 点击「开启」→ 系统权限弹窗
- 授权成功 → 进入完成页
- 拒绝 → 温和提示「你随时可以在设置中开启」→ 进入完成页（功能受限）
- 「稍后再说」→ 直接进完成页

### Step 7: 注册完成

- 大✓图标 + 撒花粒子动效
- 「欢迎加入校园足迹！」
- 1.5秒后自动跳转主页
- 或点击任意位置立即跳转

## 边界情况

| 情况 | 处理 |
|------|------|
| 注册中途退出 | 手机号+验证码步骤不保存，资料步骤暂存本地 |
| 验证码过期（5分钟） | 提交时返回「验证码过期」→重新获取 |
| 学校不在列表 | 列表底部「找不到你的学校？」→手动输入+后台审核 |
| 昵称含敏感词 | 提交时inline提示「昵称不合规，请修改」 |
| 弱网注册 | 最终提交loading较长（5-10秒），超时提示重试 |
