# 发布动态 — 交互规格

---

## 状态机

```
States:
  init            → 进入发布页，获取当前位置
  locating        → 正在获取GPS精确位置
  editing         → 位置确认，用户编辑内容
  selecting_vis   → 选择可见性规则
  selecting_user  → 定向模式：搜索/选择目标用户
  setting_time    → 定时模式：设置时间窗口
  previewing      → 预览发布效果
  submitting      → 提交中
  success         → 发布成功
  error           → 发布失败
  location_fail   → 定位失败/精度不够/不在校园内

Transitions:
  init → locating:             自动开始定位
  locating → editing:          GPS精度≤10米 + 在校园内
  locating → location_fail:    精度>10米 / 不在校园 / 超时
  location_fail → locating:    点击「重试」
  editing → selecting_vis:     点击「可见性」选择器
  selecting_vis → editing:     选择公开模式后返回
  selecting_vis → selecting_user: 选择定向/定时定向模式
  selecting_user → editing:    确认目标用户
  selecting_user → setting_time: 定时定向→设置时间
  setting_time → editing:      确认时间窗口
  editing → previewing:        点击「预览」(可选)
  editing → submitting:        点击「发布」且内容非空
  previewing → editing:        返回编辑
  previewing → submitting:     确认发布
  submitting → success:        API 200
  submitting → error:          API错误/网络错误
  error → editing:             返回修改/重试
  success → (返回地图):         1秒后自动返回

Effects:
  → locating:       顶部地图区域脉冲动画 + 「正在定位...」
  → editing:        地图定格+大头针落下弹跳 + 编辑区域滑入
  → location_fail:  全屏提示（图标+文案+重试按钮）
  → selecting_vis:  Bottom Sheet从底部滑入（300ms spring）
  → submitting:     发布按钮→spinner + 表单禁用
  → success:        按钮变✓ + 内容缩小+飘落到地图位置的粒子动效
  → error:          按钮恢复 + Toast错误提示 + 表单恢复可编辑
```

## 核心交互详解

### 1. 位置确认

- 进入时自动定位（1-3秒），地图区域脉冲等待
- 定位成功：大头针从天「落下」弹跳动效
- 大头针可拖动：拖动时显示半透明圆（20米范围限制）
- 超出20米范围：大头针弹回+轻震+提示「最多偏移20米」
- 底部显示：「📍 图书馆附近」（反向地理编码的位置名）

### 2. 内容编辑

**文字输入**：
- Placeholder：「说点什么，留在这里...」
- 右下角实时字数：「42/500」
- 接近上限时(>450)数字变橙色，到500变红+震动

**图片选择**：
- 点击「+」→ 系统相册选择器（支持多选）
- 选中后缩略图网格展示
- 长按缩略图可拖拽排序
- 点击缩略图右上「×」删除（确认弹窗：否，直接删）
- 添加第9张后「+」按钮消失

**发布按钮状态**：
- 内容为空 → 按钮灰色禁用
- 有文字或图片 → 按钮高亮可点击
- 仅有空格/换行 → 视为空，禁用

### 3. 可见性选择器（Bottom Sheet）

```
┌─────────────────────────────────┐
│ ─── (拖拽条)                     │
│                                  │
│  选择谁能看到                     │
│                                  │
│  ◉ 🌍 公开                       │
│    任何人走到这里都能看到          │
│                                  │
│  ○ 🎯 定向给TA                   │
│    只有指定的人到这里才能看到      │
│                                  │
│  ○ ⏰ 定时定向                    │
│    指定的人+指定时间才能看到       │
│                                  │
│  [确认]                          │
└─────────────────────────────────┘
```

- 选中项有高亮背景+左侧实心圆
- 切换选项时微动画（选中项展开描述文字）
- 选择「定向」→Sheet继续展开用户搜索区
- 选择「定时定向」→Sheet再展开时间选择区

### 4. 定向用户选择

- 搜索框：输入昵称/手机号尾号搜索
- 最近联系人列表（快速选择）
- 选中用户显示为Tag气泡（最多5个）
- 超过5个：提示「最多选择5个人」

### 5. 定时设置

- 开始日期+时间 / 结束日期+时间
- 快捷选项：「今天晚上」「明天上午」「本周五」
- 可选「重复」：每天/每周X/仅一次
- 时间不能早于当前时间（过去的时间灰色不可选）

## 操作清单

| 操作 | 触发 | 即时反馈 | 成功 | 失败 |
|------|------|---------|------|------|
| 拖动大头针 | drag | 实时跟手+范围圆 | 松手定位 | 超出弹回+震 |
| 添加图片 | tap「+」 | 打开相册 | 缩略图出现 | — |
| 删除图片 | tap「×」 | 图片缩小消失 | 网格重排 | — |
| 选择可见性 | tap | Sheet弹出 | 选中高亮 | — |
| 点击发布 | tap | scale+loading | 粒子飘落+返回 | Toast+恢复 |
| 取消发布 | tap「取消」 | — | 确认弹窗(有内容时) | — |

## 加载策略

| 场景 | 策略 |
|------|------|
| 进入页面 | 立即展示编辑框，地图区域脉冲等待定位 |
| 图片上传 | 选择后即开始上传（不等发布），进度条在缩略图上 |
| 搜索用户 | 300ms防抖 → 请求 → 结果列表 |
| 提交发布 | 按钮loading，2-5秒 |

## 边界情况

| 情况 | 处理 |
|------|------|
| 编辑中断网 | 允许继续编辑，发布时提示网络错误，本地暂存 |
| 编辑中退出App | 草稿自动保存，下次进入恢复 |
| 图片上传失败 | 缩略图标红+重试按钮，不阻塞其他操作 |
| 取消时有内容 | 弹窗：「放弃编辑？你的内容不会被保存」 |
| 取消时无内容 | 直接返回，不弹窗 |
| 定位一直不成功 | 15秒后提示「定位失败」+ 手动选择位置（从地标列表） |
