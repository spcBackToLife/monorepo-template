# I-09 炸弹图标 (bomb)

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | I-09 |
| 名称 | bomb |
| 类型 | Icon |
| 参考框尺寸 | 24×24 px |

### 需求来源

| 来源 | 页面 | 章节 | 需求描述 |
|------|------|------|---------|
| 页面 | 03-fishing | visual.md 3.5 | "模式切换炸弹Tab标识,表达'爆炸式大量捞取'" |

### 要解决的问题

- 功能识别: "这是炸弹模式" — 一次捞8-15人的爆发力
- 情感传达: 刺激、爆炸、大量 — 比普通撒网更刺激的付费道具
- 状态区分: 有库存(彩色醒目) vs 无库存(灰色弱化)

### 目标用户感受

> 看到就想"一发入魂"，联想到盲盒一次开很多的快感。

### 与设计系统的关系

- 引用Token: accent-gradient(#FF6B6B→#FF8C5A)有库存态 / text-tertiary(#6B6B7B)无库存态
- 同系列: 模式图标组(I-08/I-09/I-10), I-09用暖色区分"危险/爆炸"属性

---

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 简洁(4-5笔) | 24px小图标必须极简 |
| 几何 vs 有机 | 几何(圆形主体) | 经典炸弹造型=圆球 |
| 平面 vs 立体 | 平面填充 | 与Tab文字并列,不需要深度 |
| 规整 vs 随意 | 规整 | 像素对齐确保24px清晰 |

### 色彩策略

| 属性 | 值 | Token | 说明 |
|------|------|-------|------|
| 主色(有库存) | #FF6B6B→#FF8C5A | accent-gradient | 红橙=火爆/危险/刺激 |
| 主色(无库存) | #6B6B7B | text-tertiary | 灰色=不可用 |
| 火花点 | #FFB830 | gold | 金色火花=点燃感 |

### 线条特征

| 属性 | 值 | 理由 |
|------|------|------|
| 风格 | 实色填充(非描边) | 小图标填充比描边更可辨识 |
| 细节密度 | 少(3笔: 球体+导火索+火花) | 24px极简 |

### 与同系列的关系

- 与I-08(白色/主操作)和I-10(蓝色/精准)构成三色区分
- 统一点: 都是实色填充, 24px参考框(I-08是32px因为是主操作)
- 区分: 暖色调表达"危险/爆发"属性

---

## 3. 构图方案

### 参考框与安全区

```
参考框: 24×24 px
安全区: 2px
绘图区: 20×20 px
视觉重心: 居中偏下(球体)
```

### 图形描述

```
构图分层:

第1层: 炸弹球体(ellipse)
  - 形状: 圆形
  - 位置: 居中偏下(cx:12, cy:14)
  - 半径: 8px
  - 填充: #FF6B6B (accent)

第2层: 导火索(path)
  - 形状: 从球顶向右上弯曲的短弧线
  - 起点: 球顶(12, 6)
  - 终点: (16, 3)
  - 描边: #6B6B7B (深灰), 2px
  - 填充: none

第3层: 火花(2-3个小三角/星芒)
  - 位置: 导火索末端(16, 3)附近
  - 尺寸: 3-4px
  - 填充: #FFB830 (gold)
```

### 正负空间

- 正空间: ~50%(球体为主)
- 负空间: 四角
- 视觉重心: 居中(球体占主导)

---

## 4. 变体设计

| 变体名 | 场景 | 与base差异 |
|--------|------|-----------|
| available | 有库存 | 球体:#FF6B6B, 火花:#FFB830(完整彩色) |
| unavailable | 无库存(数量=0) | 全部统一为text-tertiary(#6B6B7B),无金色火花 |

### 变体详细规格

#### available (有库存)

| 属性 | 值 | 理由 |
|------|------|------|
| 球体色 | #FF6B6B | accent红=醒目=有 |
| 导火索 | #4A4A58 | 深灰 |
| 火花 | #FFB830 | 金色=点燃状态 |

#### unavailable (无库存)

| 属性 | 值 | 理由 |
|------|------|------|
| 全部颜色 | #6B6B7B | text-tertiary统一灰=不可用 |
| 火花 | 不显示或同灰 | 未点燃=没有 |

### 变体间过渡

- 无动画过渡(库存变化为数据刷新,非交互触发)

---

## 5. 应用效果

### 最终视觉

- 在底部模式切换Tab中,与"炸弹"文字左侧并列
- 有库存时红色+金色火花极度醒目(暗色底上)
- 无库存时灰色融入背景,不干扰视线

### 应用后附加样式

```css
width: 24px;
height: 24px;
/* 无额外效果 */
```

### 应用技术方案

| 步骤 | 操作 |
|------|------|
| 1 | Tab内创建24×24 icon div |
| 2 | material-painter绘制炸弹 |
| 3 | export_and_apply |
| 4 | 清理background |

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 24×24 px |
| 背景 | transparent |
| 目标节点 | 炸弹模式Tab 内的 24×24 icon div |

### 图形描述

```
一个经典炸弹图标，传达"爆炸式大量捞取"的刺激感。

组成元素：
1. 炸弹球体: 实心圆形，直径约16px
   - 位于参考框中心偏下
   - 填充: #FF6B6B (accent红)

2. 导火索: 从球顶向右上弯曲的短弧线
   - 长约6-8px
   - 描边: #4A4A58 (深灰), 2px

3. 火花: 导火索末端2-3条短放射线
   - 长约3-4px
   - 描边: #FFB830 (gold), 1.5px
   - 表达"已点燃"的状态

视觉重心居中偏下（球体为主）。
```

### 要求清单

- 色彩: 三色组合 — 红球体(#FF6B6B) + 灰导火索(#4A4A58) + 金火花(#FFB830)
- 风格: 球体实心填充 + 导火索/火花线性描边
- 对齐: 球体居中偏下，导火索向右上延伸
- 同系列: 与 I-10(蓝色精准) 构成冷暖对比

### 质量标准

- [ ] 球体圆润居中
- [ ] 导火索短而自然(不超出安全区)
- [ ] 火花金色醒目
- [ ] 24px 下整体可辨识
- [ ] 透明通道正确
- [ ] 暗色底上 accent 红醒目
