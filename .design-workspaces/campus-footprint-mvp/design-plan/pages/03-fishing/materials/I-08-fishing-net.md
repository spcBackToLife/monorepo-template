# I-08 渔网图标 (fishing-net)

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | I-08 |
| 名称 | fishing-net |
| 类型 | Icon |
| 参考框尺寸 | 32×32 px |

### 需求来源

| 来源 | 页面 | 章节 | 需求描述 |
|------|------|------|---------|
| 页面 | 03-fishing | visual.md 3.5 | "FAB按钮内撒网图标,表达核心操作'撒网捞人'" |

### 要解决的问题

- 功能识别: "这是撒网按钮" — 一看就知道是"捞人"操作
- 品牌强化: 渔网形态强化"捞人"隐喻,区别于通用搜索/刷新图标
- 情感传达: "向水面撒出一张网" — 动作感+期待感

### 目标用户感受

> 看到就联想到"撒网捞鱼"的动作，产生"试试手气"的冲动。

### 与设计系统的关系

- 引用Token: text-primary(#F2F2F7) 填充 — 在secondary渐变按钮上保证高对比
- 遵循规则: 实色填充(因为是FAB内核心图标,激活态规范)
- 同系列: 与I-09炸弹/I-10准星同为模式标识图标族(但I-08尺寸更大=主操作)

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 简洁(5-7笔) | FAB内32px图标需要一眼认出,细节太多糊成一团 |
| 几何 vs 有机 | 几何+有机混合 | 网格=几何规律, 网兜弧线=有机柔和 |
| 平面 vs 立体 | 平面 | 在渐变按钮上,图标保持平面避免视觉冲突 |
| 规整 vs 随意 | 规整 | 图标需要像素级精确,传达可靠感 |

### 色彩策略

| 属性 | 值 | Token | 说明 |
|------|------|-------|------|
| 主色 | #F2F2F7 | text-primary | 白色在紫蓝渐变上最醒目 |
| 辅色 | 无 | — | 单色保持简洁 |
| 渐变 | 否 | — | 按钮本身已是渐变,图标保持纯色 |

### 线条特征

| 属性 | 值 | 理由 |
|------|------|------|
| 线宽 | 2px | 32px图标用2px确保清晰 |
| 端点 | round | 柔和,与圆角系统统一 |
| 拐角 | round | 网绳不是直角 |
| 闭合性 | 混合(弧线开放+网格闭合) | 自然的网形态 |
| 细节密度 | 中(7笔) | 弧+手柄+3×2网格线 |

### 与同系列的关系

- 系列: "捞人模式图标组"(I-08/I-09/I-10)
- I-08最大(32px)=主操作, I-09/I-10=24px=次要选项
- 统一点: 都是实色填充, 都有明确的隐喻形态
- 区分: I-08白色(在渐变FAB上), I-09暖色(炸弹), I-10蓝色(精准)

## 3. 构图方案

### 参考框与安全区

```
参考框: 32×32 px
安全区: 2px
绘图区: 28×28 px
视觉重心: 偏下(网兜在下方=重力感)
```

### 图形描述

```
构图分层:

第1层: 网兜主体(path)
  - 形状: 半圆弧(开口朝上) + 底部收口
  - 尺寸: 约24×16px
  - 位置: 居中偏下(y: 14-30)
  - 描边: #F2F2F7, 2px
  - 填充: none(仅描边)

第2层: 网格线(竖×3 + 横×2)
  - 竖线3条: 等分弧线, 从弧顶到底部收口
  - 横线2条: 在弧线1/3和2/3高度处的弧线
  - 描边: #F2F2F7, 1.5px
  - 端点: round

第3层: 手柄(line)
  - 形状: 一条斜线从网兜左上方向右上延伸
  - 位置: 从弧线左端点(约 x:4,y:14)到右上(约 x:28,y:4)
  - 描边: #F2F2F7, 2px
  - 端点: round
```

### 正负空间

- 正空间(线条面积): ~35%
- 负空间: 网格间隙(透出按钮渐变色=呼吸感)
- 视觉重心: 偏下60%(网兜向下,手柄向上 = 动态平衡)

## 4. 变体设计

| 变体名 | 场景 | 与base差异 |
|--------|------|-----------|
| idle | FAB正常态 | 标准(如上) |
| casting | 撒网动效中 | 不使用此图标(FAB内替换为文字"收网中...") |

无多变体需求 — 图标仅在idle态显示,casting态被文字替换。

## 5. 应用效果

### 最终视觉

- 在80×80紫蓝渐变FAB按钮正中心,白色渔网图标清晰可辨
- 周围有脉冲glow扩散(按钮效果,非图标自身)
- 32px尺寸在80px按钮中占40%,留有充足呼吸空间

### 应用后附加样式

```css
/* icon容器 */
width: 32px;
height: 32px;
/* 无额外filter/shadow — 按钮glow已足够 */
```

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 32×32 px |
| 背景 | transparent |
| 目标节点 | cast-btn 内的 32×32 icon div |

### 图形描述

```
一个撒网/渔网图标，传达"向水面撒出一张网"的动作感。

组成元素：
1. 网兜主体: 半圆弧形(开口朝上)，底部收口形成兜状
   - 宽约24px，高约16px，位于参考框中下部
   - 仅描边，2px线宽，round端点

2. 网格线: 3条竖线 + 2条横弧线
   - 竖线等分网兜，从弧顶到底部收口
   - 横弧线在网兜1/3和2/3高度
   - 1.5px线宽

3. 手柄: 一条从网兜左上方斜向右上方延伸的线
   - 表达"手持撒出"的动态
   - 2px线宽

全部白色 #F2F2F7，在紫蓝渐变按钮上高对比。
视觉重心偏下(网兜向下=重力感)。
```

### 要求清单

- 色彩: 单色白 #F2F2F7 (text-primary)
- 线宽: 主轮廓2px + 网格线1.5px
- 风格: 线性描边，round 端点/拐角
- 对齐: 水平居中，垂直偏下
- 同系列: 与 I-09/I-10 为模式图标组，I-08 最大(32px)=主操作

### 质量标准

- [ ] 网兜形态自然(弧线+收口)
- [ ] 网格线均匀分布
- [ ] 手柄与网兜连接自然
- [ ] 2px 线宽在 32px 尺寸下清晰
- [ ] 白色在渐变按钮上高对比
- [ ] 透明通道正确(无白底)
