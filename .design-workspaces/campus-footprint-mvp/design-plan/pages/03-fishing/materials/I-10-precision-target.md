# I-10 精准准星图标 (precision-target)

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | I-10 |
| 名称 | precision-target |
| 类型 | Icon |
| 参考框尺寸 | 24×24 px |

### 需求来源

| 来源 | 页面 | 章节 | 需求描述 |
|------|------|------|---------|
| 页面 | 03-fishing | visual.md 3.5 | "模式切换精准Tab标识,表达'精准定向捞取'" |

### 要解决的问题

- 功能识别: "这是精准模式" — 设置条件后精准匹配
- 情感传达: 精确、专注、科技感 — 有目的地寻找
- 状态区分: 有库存(primary色) vs 无库存(灰色)

### 目标用户感受

> 看到准星想到"瞄准目标"，期待精准找到那个对的人。

### 与设计系统的关系

- 引用Token: primary(#4F8CFF)有库存态 / text-tertiary(#6B6B7B)无库存态
- 遵循规则: 线性1.5px/round端点(标准图标规范)
- 同系列: 模式图标组(I-08/I-09/I-10), I-10用蓝色表达科技/精准

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 简洁(5-6笔) | 24px线性图标 |
| 几何 vs 有机 | 纯几何(圆+线) | 准星/瞄准器=纯几何形态 |
| 平面 vs 立体 | 平面 | 线性图标不做立体 |
| 规整 vs 随意 | 极度规整(对称) | 准星必须精确对称 |

### 色彩策略

| 属性 | 值 | Token | 说明 |
|------|------|-------|------|
| 主色(有库存) | #4F8CFF | primary | 蓝色=科技/精准/冷静 |
| 主色(无库存) | #6B6B7B | text-tertiary | 灰色=不可用 |

### 线条特征

| 属性 | 值 | 理由 |
|------|------|------|
| 线宽 | 1.5px | 20-24px图标标准线宽 |
| 端点 | round | 全局统一 |
| 拐角 | — | 无拐角(圆+直线) |
| 闭合性 | 混合(圆环闭合+刻度线开放) | 准星标准形态 |
| 细节密度 | 中(外圆+内圆+4刻度) | 准星最小识别要素 |

### 与同系列的关系

- 与I-09(红色/爆发)构成冷暖对比
- 统一点: 24px, 两个变体(有/无库存)
- 区分: 线性描边(vs I-09填充) — 因为精准模式"冷静克制"

## 3. 构图方案

### 参考框与安全区

```
参考框: 24×24 px
安全区: 2px
绘图区: 20×20 px
视觉重心: 正中(完全对称)
```

### 图形描述

```
构图分层:

第1层: 外圆环(circle)
  - 形状: 圆环(仅描边)
  - 位置: 正中(cx:12, cy:12)
  - 半径: 9px
  - 描边: #4F8CFF, 1.5px
  - 填充: none

第2层: 内圆点(circle)
  - 形状: 小实心圆
  - 位置: 正中(cx:12, cy:12)
  - 半径: 2px
  - 填充: #4F8CFF

第3层: 十字刻度线(4条line)
  - 上: (12,2)→(12,5)
  - 下: (12,19)→(12,22)
  - 左: (2,12)→(5,12)
  - 右: (19,12)→(22,12)
  - 描边: #4F8CFF, 1.5px
  - 端点: round
```

### 正负空间

- 正空间(线条): ~20%(线性图标空间利用率低)
- 负空间: 大量(圆环内外都是空)
- 视觉重心: 正中心(完全对称=平衡稳定)

## 4. 变体设计

| 变体名 | 场景 | 与base差异 |
|--------|------|-----------|
| available | 有库存 | 全部#4F8CFF(primary蓝) |
| unavailable | 无库存 | 全部#6B6B7B(text-tertiary灰) |

### 变体间过渡

- 无动画(数据驱动切换,非交互触发)

## 5. 应用效果

### 最终视觉

- 在底部模式切换Tab中,"精准"文字左侧
- 有库存时primary蓝色在暗底上清晰(科技感)
- 无库存时灰色退入背景

### 应用后附加样式

```css
width: 24px;
height: 24px;
```

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 24×24 px |
| 背景 | transparent |
| 目标节点 | 精准模式Tab 内的 24×24 icon div |

### 图形描述

```
一个瞄准器/准星图标，完全对称的几何构图。

组成元素：
1. 外圆环: 直径约18px，正中，仅描边无填充
2. 中心圆点: 直径约4px，正中，实心填充
3. 十字刻度线: 上下左右各一条约3px短线
   - 从外圆环外边缘向外延伸
   - 与圆环有约2px间距（不粘连）

全部 #4F8CFF (primary蓝)，1.5px 线宽，round 端点。
完全四向对称，视觉重心正中。
```

### 要求清单

- 色彩: 单色 #4F8CFF (primary)
- 线宽: 1.5px，24px 图标标准
- 风格: 线性描边 + 中心实心点
- 对齐: 完全对称（准星必须精确）
- 同系列: 与 I-09(红色爆发) 构成冷暖对比

### 质量标准

- [ ] 完全对称(四向刻度等长等距)
- [ ] 外圆环与刻度线有间距(不粘连)
- [ ] 1.5px 线宽在 24px 下清晰
- [ ] 中心圆点大小适中(辨识+不突兀)
- [ ] primary 蓝在暗底上对比度足够
- [ ] 透明通道正确
