# I-09 定时可见性图标 (vis-timed)

---

## 1. 设计意图

| 字段 | 值 |
|------|------|
| 编号 | I-09 |
| 名称 | vis-timed |
| 类型 | Icon |
| 参考框尺寸 | 20×20 px |

### 需求来源

| 来源 | 组件 | 需求描述 |
|------|------|---------|
| 组件 | VisibilitySheet | "识别定时定向模式" |

### 要解决的问题

- 隐喻传达: 时钟=时间限制=定时触发

### 与设计系统的关系

- 系列: 可见性三件套(I-07/08/09)
- 统一规范同上

---

## 2. 风格分析

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 | 3笔(圆+时针+分针) | 时钟最经典表达 |
| 几何 | 精确圆+直线 | 匹配"精确时间" |

---

## 3. 构图方案

```
参考框: 20×20, 绘图区: 16×16

图形:
  - 表盘圆: 直径14px，居中，仅描边
  - 时针: 从圆心向上(12点方向)，长约4px
  - 分针: 从圆心向右(3点方向)，长约3px
  → 时分针呈L形(12:15位置), 简洁清晰
```

---

## 4. 变体设计

| 变体 | 颜色 |
|------|------|
| inactive | text-secondary (#A8A8B3) |
| active | primary (#4F8CFF) |

---

## 5. 应用效果

- VisibilitySheet第3选项行左侧
- 与文字"定时定向"配合

---

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 20×20 px |
| 背景 | transparent |
| 目标节点 | VisibilitySheet 定时选项行的 20×20 icon div |

### 图形描述

```
一个简洁的时钟图标，表达"定时触发/时间限制"。

组成元素：
1. 表盘圆: 直径约14px，居中，仅描边
2. 时针: 从圆心向正上方(12点)，长约4px
3. 分针: 从圆心向正右方(3点)，长约3px

时针和分针从同一圆心出发，呈L形(显示12:15)。
全部 #A8A8B3 (text-secondary)，1.5px 线宽，round 端点。
```

### 要求清单

- 色彩: 单色 #A8A8B3 (text-secondary)
- 线宽: 1.5px
- 风格: 线性描边
- 对齐: 表盘居中，指针从圆心出发
- 同系列: 可见性三件套统一风格

### 质量标准

- [ ] 表盘圆居中
- [ ] 时针/分针从圆心出发，方向正确
- [ ] 与 I-07/I-08 视觉重量平衡
- [ ] 透明通道正确
