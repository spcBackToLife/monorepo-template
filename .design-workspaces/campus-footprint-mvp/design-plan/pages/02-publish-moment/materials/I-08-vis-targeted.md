# I-08 定向可见性图标 (vis-targeted)

---

## 1. 设计意图

| 字段 | 值 |
|------|------|
| 编号 | I-08 |
| 名称 | vis-targeted |
| 类型 | Icon |
| 参考框尺寸 | 20×20 px |

### 需求来源

| 来源 | 组件 | 需求描述 |
|------|------|---------|
| 组件 | VisibilitySheet | "识别定向可见性模式" |

### 要解决的问题

- 隐喻传达: 靶心=精确指向=只给特定人看

### 与设计系统的关系

- 系列: 可见性三件套(I-07/08/09)
- 统一: 1.5px线宽, round端点, 20×20

---

## 2. 风格分析

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 | 4笔(双圆环+中心点+十字线) | 辨识度 |
| 几何 | 同心圆=精确性 | 匹配"精准定向"含义 |

---

## 3. 构图方案

```
参考框: 20×20, 绘图区: 16×16

图形:
  - 外圆: 直径14px，居中，仅描边
  - 内圆: 直径7px，居中，仅描边
  - 中心点: 直径3px，居中，实心填充
  - (可选十字线: 上下左右各2px短线从外圆边缘延伸)
```

---

## 4. 变体设计

| 变体 | 颜色 |
|------|------|
| inactive | text-secondary (#A8A8B3) |
| active | primary (#4F8CFF) |

---

## 5. 应用效果

- VisibilitySheet第2选项行左侧
- 与文字"定向给TA"配合

---

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 20×20 px |
| 背景 | transparent |
| 目标节点 | VisibilitySheet 定向选项行的 20×20 icon div |

### 图形描述

```
一个靶心/同心圆图标，表达"精准定向=只给特定人看"。

组成元素：
1. 外圆环: 直径约14px，居中，仅描边
2. 内圆环: 直径约7px，居中，仅描边
3. 中心实心点: 直径约3px，居中，实心填充

全部 #A8A8B3 (text-secondary)，1.5px 线宽。
三层同心圆递进收缩，传达"精准锁定"的含义。
```

### 要求清单

- 色彩: 单色 #A8A8B3 (text-secondary)
- 线宽: 1.5px
- 风格: 线性描边(两圆环) + 实心(中心点)
- 对齐: 完全居中对称
- 同系列: 可见性三件套统一风格

### 质量标准

- [ ] 三层同心圆完全对称
- [ ] 间距均匀（外圆→内圆→中心点）
- [ ] 与 I-07/I-09 视觉重量平衡
- [ ] 透明通道正确
