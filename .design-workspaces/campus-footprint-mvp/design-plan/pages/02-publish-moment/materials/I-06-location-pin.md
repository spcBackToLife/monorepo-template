# I-06 定位大头针 (location-pin)

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | I-06 |
| 名称 | location-pin |
| 类型 | Icon |
| 参考框尺寸 | 24×32 px |

### 需求来源

| 来源 | 页面 | 章节 | 需求描述 |
|------|------|------|---------|
| 页面 | 02-publish-moment | visual.md 3.5 | "地图上标识发布位置,品牌化水滴形" |

### 要解决的问题

- 功能识别: "这是我要发布动态的位置"
- 品牌强化: 定制Pin区别于系统默认地图标记
- 情感传达: "钉住了，这条动态留在这里"

### 目标用户感受

> 看到Pin落地时感到确定——"好，就留在这里了"。

### 与设计系统的关系

- 引用Token: primary(#4F8CFF)填充, text-primary(#F2F2F7)内圆
- 这是品牌化元素，在多个页面地图中会出现(发布/预览/个人动态)
- 同系列: 地图标记组(I-06 + 01-home-map的气泡是同一视觉家族)

---

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 简洁(2层) | 地图上需要一眼认出 |
| 几何 vs 有机 | 有机(水滴形) | 与品牌"足迹/位置"的柔和感匹配 |
| 平面 vs 立体 | 略微立体(内圆暗示深度) | 在平面地图上需要"突出"感 |

### 色彩策略

| 属性 | 值 | Token | 说明 |
|------|------|-------|------|
| 主体填充 | #4F8CFF | primary | 品牌色,暗色底上醒目 |
| 内圆 | #F2F2F7 | text-primary | 高对比中心点 |
| 阴影 | rgba(79,140,255,0.3) | — | 微弱glow暗示"这个很重要" |

### 与同系列的关系

- 与MapBubble同为"地图上的品牌标记"
- Pin=用户主动放置(发布位置), Bubble=被动展示(他人动态)
- 统一点: 都使用primary色, 都有glow, 都是圆润形态

---

## 3. 构图方案

### 参考框与安全区

```
参考框: 24×32 px (比标准方形高——水滴形需要纵向空间)
安全区: 2px
绘图区: 20×28 px
视觉重心: 偏上(Pin的"头"在上部)
```

### 图形描述

```
构图分层:

第1层: 水滴形主体(path)
  - 形状: 上方圆润(r≈9px), 下方收尖
  - 尺寸: 约18×24px
  - 位置: 居中, 底部尖端在y≈28
  - 填充: #4F8CFF (primary)

第2层: 内圆高光
  - 形状: circle
  - 位置: Pin头部中心(约 cx:12, cy:12)
  - 半径: 4px
  - 填充: #F2F2F7 (白色)
```

### 正负空间

- 正空间: ~55% (水滴形填满大部分)
- 负空间: 四角(水滴形不到的区域)
- 视觉重心: 上部2/3(Pin的圆头)

---

## 4. 变体设计

| 变体名 | 场景 | 差异 |
|--------|------|------|
| locating | 定位中(脉冲态) | opacity:0.6 + 缩小scale(0.8) |
| located | 已定位(落地后) | opacity:1 + 完整尺寸 + 底部接触阴影 |

### 变体间过渡

- locating → located: 500ms spring (从上落下+弹跳)
- opacity: 0.6→1 (200ms ease-out)

---

## 5. 应用效果

### 最终视觉

- 地图预览区中心，作为用户当前位置的"锚点"
- 底部有小接触阴影(暗示Pin"站"在地面上)
- 可被用户拖动(拖动时scale→1.1,松手scale→1)

### 应用后附加样式

```css
/* Pin容器 */
width: 24px;
height: 32px;
filter: drop-shadow(0 2px 4px rgba(0,0,0,0.4));
transition: transform 200ms ease-out;
```

### 应用技术方案

| 步骤 | 操作 |
|------|------|
| 1 | 地图标记容器内创建24×32 icon div |
| 2 | material-painter绘制水滴形+内圆 |
| 3 | export_and_apply |
| 4 | 清理background |

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 24×32 px |
| 背景 | transparent |
| 目标节点 | 地图标记容器内的 24×32 icon div |

### 图形描述

```
一个经典的地图定位 Pin（水滴形），表达用户当前位置。

组成元素：
1. 水滴形主体: 上圆下尖的经典Pin形状
   - 宽约20px，高约28px，居中
   - 填充: #4F8CFF (primary)
   - 无描边

2. 内圆白点: Pin头部(圆形部分)正中
   - 直径约8px
   - 填充: #F2F2F7 (白色)
   - 增加层次感和识别度

视觉重心偏上(圆形头部)，底部收尖指向地面。
```

### 要求清单

- 色彩: primary蓝填充 + 白色内圆
- 风格: 实心填充，无描边
- 对齐: 水平居中，注意非方形参考框(24×32)
- 形态: 经典水滴/Pin形，对称，底部收尖自然

### 质量标准

- [ ] 水滴形对称，底部收尖自然
- [ ] 内圆在 Pin 头部居中
- [ ] primary 填充在暗底上醒目
- [ ] 透明通道正确
- [ ] 24×32 非方形参考框设置正确
