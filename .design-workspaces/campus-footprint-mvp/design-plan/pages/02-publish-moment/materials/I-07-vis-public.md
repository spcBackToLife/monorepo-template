# I-07 公开可见性图标 (vis-public)

---

## 1. 设计意图

| 字段 | 值 |
|------|------|
| 编号 | I-07 |
| 名称 | vis-public |
| 类型 | Icon |
| 参考框尺寸 | 20×20 px |

### 需求来源

| 来源 | 组件 | 需求描述 |
|------|------|---------|
| 组件 | VisibilitySheet | "识别公开可见性模式" |

### 要解决的问题

- 功能识别: 一眼区分「公开」与其他两种模式
- 隐喻传达: 地球=所有人=公开

### 与设计系统的关系

- 线宽: 1.5px, 端点round
- 系列: 可见性图标三件套(I-07/I-08/I-09), 视觉重量必须平衡
- 颜色: text-secondary(未选中) / primary(选中)

---

## 2. 风格分析

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 简洁(4笔) | 20px小尺寸 |
| 几何 vs 有机 | 几何(圆+线) | 地球=精确/通用 |

### 线条特征

| 属性 | 值 |
|------|------|
| 线宽 | 1.5px |
| 端点 | round |
| 细节 | 4笔: 外圆+横线+竖弧线+横弧线 |

---

## 3. 构图方案

```
参考框: 20×20, 安全区: 2px, 绘图区: 16×16

图形描述:
  - 外圆: cx:10, cy:10, r:7, stroke 1.5px
  - 中横线(赤道): (3,10) → (17,10), stroke 1.5px
  - 竖弧(经线): 椭圆弧通过(10,3)和(10,17), rx:4
  - 横弧(纬线): 椭圆弧在y:6位置, 微弯
```

---

## 4. 变体设计

| 变体 | 颜色 | 附加 |
|------|------|------|
| inactive | text-secondary (#A8A8B3) | 无 |
| active | primary (#4F8CFF) | 无(颜色变化即可) |

---

## 5. 应用效果

- VisibilitySheet选项行左侧
- 与文字"公开"配合
- 20×20在64px高选项行中垂直居中

---

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 20×20 px |
| 背景 | transparent |
| 目标节点 | VisibilitySheet 公开选项行的 20×20 icon div |

### 图形描述

```
一个地球仪图标，表达"公开可见=全世界能看"。

组成元素：
1. 外圆(地球轮廓): 直径约14px，居中，仅描边
2. 赤道横线: 穿过圆心的水平线
3. 经线竖弧: 穿过南北极的纵向椭圆弧线

全部 #A8A8B3 (text-secondary)，1.5px 线宽。
传达"公开/全球"的含义。
```

### 要求清单

- 色彩: 单色 #A8A8B3 (text-secondary)
- 线宽: 1.5px
- 风格: 线性描边
- 对齐: 居中
- 同系列: 可见性三件套(I-07/08/09)统一风格

### 质量标准

- [ ] 地球形态可辨识
- [ ] 线宽统一 1.5px
- [ ] 与 I-08/I-09 视觉重量平衡
- [ ] 透明通道正确
