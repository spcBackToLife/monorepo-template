# VisibilitySheet — 视觉设计分析

> 层级: L2 组件深钻 | 上级: pages/02-publish-moment/visual.md | 角色: 配角-设置

---

## 1. 情感与氛围目标

### 1.1 情感定位

| 维度 | 回答 |
|------|------|
| 用户此刻的心理状态 | 已写完内容，在做最后的分发决策——"谁能看到我写的？" |
| 目标感受 | **清晰、掌控、安全** — 我完全知道谁能看到、为什么选这个 |
| 情绪曲线 | 浏览选项(平静) → 理解差异(认知清晰) → 做出选择(掌控感) |
| 与主题风格的关系 | 暗色毛玻璃Sheet保持午夜调性，但内容高对比确保可读 |
| 页面给我的视觉预算 | 角色:配角-设置, 权重:5/10, 允许:毛玻璃+图标+选中高亮, 装饰上限:少 |

### 1.2 品牌感要素

- 三种可见性模式的**图标差异化** — 一眼区分公开/定向/定时
- 选中态的primary高亮 — 品牌色在功能场景的克制运用

---

## 2. 视觉层级设计

### 2.1 空间深度(组件内)

```
[底层] ─── 毛玻璃背景(遮盖发布页)
    ↑
[内容层] ─── 标题 + 选项列表(核心)
    ↑
[强调层] ─── 选中项高亮背景 + 确认按钮
```

### 2.2 视觉权重(组件内)

| 元素 | 权重 | 手段 | 为什么 |
|------|:-:|------|--------|
| 选项图标 | 7 | 20px图标+色彩区分(选中primary/未选中灰) | 快速识别模式类型 |
| 选项文字 | 6 | body-md主文字+body-sm描述 | 理解每个选项 |
| 选中指示器 | 5 | 左侧实心/空心圆+右侧背景高亮 | 当前状态反馈 |
| 确认按钮 | 7 | primary-gradient(有选择时) | 完成操作 |
| 标题 | 4 | heading-md | 上下文 |

## 3. 视觉手段清单

### 3.1 色彩

| 手段 | 描述 | 感受 | Token |
|------|------|------|-------|
| 选中项背景 | primary at 10% | 柔和高亮 | primary at 10% |
| 选中图标色 | primary #4F8CFF | 激活态 | primary |
| 未选中图标色 | text-tertiary | 弱化 | text-tertiary |
| 选中Radio圆 | primary填充 | 确定性 | primary |
| 未选Radio圆 | 空心+border | 可选择 | text-tertiary border |

### 3.2 光影

| 手段 | 对象 | 参数 |
|------|------|------|
| Sheet整体阴影 | 容器 | 0 -8px 32px rgba(0,0,0,0.5) |
| 顶部边框光 | 顶边 | 1px solid rgba(255,255,255,0.08) |

### 3.3 图标(素材需求)

| 图标 | 场景 | 素材ID |
|------|------|--------|
| 地球(公开) | 第1选项 | I-07 |
| 靶心(定向) | 第2选项 | I-08 |
| 时钟(定时) | 第3选项 | I-09 |

## 4. 实现分类

| # | 元素 | 分类 | 理由 |
|---|------|:-:|------|
| 1 | 毛玻璃背景 | CSS | backdrop-filter |
| 2 | 选中项高亮背景 | CSS | background-color |
| 3 | Radio圆形指示器 | CSS | border-radius+background |
| 4 | **公开图标(地球)** | 素材 | 多笔画(圆+经纬线) |
| 5 | **定向图标(靶心)** | 素材 | 多笔画(同心圆+中心点) |
| 6 | **定时图标(时钟)** | 素材 | 多笔画(圆+时分针) |
| 7 | Sheet弹出动效 | CSS动效 | translateY transition |

## 5. 素材需求清单

| 素材ID | 名称 | 意图 | 尺寸 | 色彩 |
|--------|------|------|------|------|
| I-07 | vis-public | 「所有人可见」 | 20×20 | text-secondary/primary |
| I-08 | vis-targeted | 「指定人可见」 | 20×20 | text-secondary/primary |
| I-09 | vis-timed | 「限时指定人」 | 20×20 | text-secondary/primary |

## 6. 样式规格

| 属性 | 值 | 为什么 |
|------|------|--------|
| background | rgba(20,20,32,0.9) | 更不透明(设置面板需要高对比) |
| backdrop-filter | blur(20px) saturate(1.2) | 统一毛玻璃 |
| border-radius | 24px 24px 0 0 | Sheet大圆角 |
| padding | 20px 20px 34px | 内容呼吸+安全区 |
| 选项高度 | 64px | 舒适触摸+内容承载 |
| 选项间距 | 0(紧密)+divider分隔 | 选项组紧凑 |
| 选中背景 | rgba(79,140,255,0.1) | primary at 10%柔和 |
| 选中圆角 | 12px (radius-md) | 柔和高亮区域 |

## 7. 风格一致性检查

| 检查项 | 回答 |
|--------|------|
| 色彩来自Token？ | ✅ primary/text-secondary/text-tertiary/Layer相关 |
| 毛玻璃参数统一？ | ✅ 与NavBar/PreviewCard同参数 |
| 图标风格统一？ | ✅ 线性1.5px/round(I-07~09与系列一致) |
| 预算内？ | ✅ 权重5,无额外装饰,仅功能性图标 |
