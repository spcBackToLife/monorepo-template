# PreviewCard — 视觉设计分析

> 层级: L2 组件深钻 | 上级: pages/01-home-map/visual.md | 角色: 配角-信息

---

## 1. 情感与氛围目标

### 1.1 情感定位

| 维度 | 回答 |
|------|------|
| 用户此刻的心理状态 | 已点击气泡，带着好奇心等待内容揭示 |
| 目标感受 | **清晰、有序、可操作** — 快速获取信息并决定是否深入 |
| 情绪曲线 | 期待(弹出) → 扫读(头像/文字/图片) → 决策(看详情或关闭) |
| 与主题风格的关系 | 毛玻璃+暗色调保持午夜感，但内容清晰度优先于装饰 |
| 页面给我的视觉预算 | 角色:配角-信息, 权重:5/10, 允许:毛玻璃+阴影+圆角, 装饰上限:少 |

### 1.2 品牌感要素

- 毛玻璃底+大圆角顶部 = App的Sheet视觉语言统一
- 信息布局(头像+文字+缩略图+交互)是社交产品通用但配色/圆角保持品牌感

---

## 2. 视觉层级设计

### 2.1 空间深度(组件内)

```
[底层] ─── 毛玻璃背景(模糊+暗透明) — 与地图内容柔和分隔
    ↑
[结构层] ─── 拖拽条/分隔线(极微弱白线) — 暗示可操作
    ↑
[内容层] ─── 头像/文字/缩略图/按钮 — 信息主体
```

### 2.2 视觉权重(组件内)

| 元素 | 权重(1-10) | 手段 | 为什么 |
|------|:-:|------|--------|
| 动态文字 | 7 | body-lg + text-primary | 最重要的内容 |
| 缩略图 | 6 | 圆角图片+适中尺寸 | 视觉吸引力 |
| 头像+用户名 | 5 | 圆形头像+body-sm | 社交身份信息 |
| 查看详情按钮 | 5 | primary色文字+箭头 | 行动引导 |
| 互动数据(❤️💬) | 3 | caption+text-tertiary | 辅助信息 |
| 拖拽条 | 1 | 极细小横条 | 操作暗示 |

## 3. 视觉手段清单

### 3.1 色彩运用

| 色彩手段 | 描述 | 面积 | 感受 | Token |
|---------|------|------|------|-------|
| 毛玻璃底 | rgba(20,20,32,0.85) + blur | 整个卡片 | 层级分明/通透 | Layer1 at 85% |
| 白色主文字 | text-primary | 标题/正文 | 可读性第一 | text-primary |
| 灰色辅助 | text-tertiary | 时间/计数 | 弱化非核心 | text-tertiary |
| Primary色链接 | #4F8CFF | 查看详情 | 可交互暗示 | primary |

### 3.2 光影效果

| 光影 | 描述 | 对象 | 感受 | 参数 |
|------|------|------|------|------|
| 上投影 | 向上扩散的深色阴影 | 整个卡片 | 悬浮于地图之上 | 0 -8px 32px rgba(0,0,0,0.5) |
| 顶部边框光 | 极微弱白色上边框 | 卡片顶部 | 玻璃边缘反光 | border-top: 1px solid rgba(255,255,255,0.08) |

### 3.3 动效

| 动效 | 触发 | 效果 | 感受 | 参数 |
|------|------|------|------|------|
| 弹出 | 点击气泡 | translateY(100%→0) | 信息浮出 | 300ms ease-out |
| 收起 | 下滑/点击空白 | translateY(0→100%) | 自然消失 | 250ms ease-in |
| 展开 | 上滑卡片 | height(30%→60%) | 想看更多 | 300ms ease-out |

## 4. 实现分类

| # | 元素 | 分类 | 理由 |
|---|------|:-:|------|
| 1 | 毛玻璃背景 | CSS | backdrop-filter完美实现 |
| 2 | 阴影+边框 | CSS | box-shadow+border |
| 3 | 弹出/收起动效 | CSS | transform transition |
| 4 | 头像圆形 | CSS | border-radius:50% |
| 5 | 缩略图圆角 | CSS | border-radius: radius-sm |
| 6 | 查看详情箭头 | 素材 | 与系列图标统一(I-04) |

**结论: 仅「查看详情箭头」需要素材(I-04)，其余CSS实现。**

## 6. 样式规格

| 属性 | 值 | 为什么 |
|------|------|--------|
| background | rgba(20,20,32,0.85) | Layer1透明底毛玻璃 |
| backdrop-filter | blur(20px) saturate(1.2) | 通透层级感 |
| border-radius | 24px 24px 0 0 | 顶部大圆角=Sheet |
| box-shadow | 0 -8px 32px rgba(0,0,0,0.5) | 悬浮于地图之上 |
| border-top | 1px solid rgba(255,255,255,0.08) | 玻璃边缘反光 |
| padding | 12px 20px 24px | 内容呼吸空间 |
| max-height | 30vh (默认) / 60vh (展开) | Sheet高度控制 |

## 7. 风格一致性检查

| 检查项 | 回答 |
|--------|------|
| 色彩来自Token？ | ✅ Layer1/text-primary/text-tertiary/primary |
| 毛玻璃参数与NavBar一致？ | ✅ 同参数 blur:20px, saturate:1.2 |
| 圆角引用标准Token？ | ✅ radius-xl(24px) |
| 装饰在预算内？ | ✅ 无额外装饰，仅结构性光影 |
| 动效时长标准？ | ✅ 300ms/250ms ease-out/ease-in |
