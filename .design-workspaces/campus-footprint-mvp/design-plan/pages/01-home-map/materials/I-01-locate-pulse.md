# I-01 定位回位图标 (locate-pulse)

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | I-01 |
| 名称 | locate-pulse |
| 类型 | Icon |
| 参考框尺寸 | 20×20 px |

### 需求来源

| 来源 | 页面/组件 | 章节 | 需求描述 |
|------|---------|------|---------|
| 页面 | 01-home-map | visual.md 3.5 | "定位回位图标，回到当前位置" |

### 要解决的问题

- 功能识别: 用户一眼识别「点这个可以回到我的位置」
- 操作暗示: 在地图被拖动后，给用户「回到原点」的安心感

### 目标用户感受

> 看到这个图标时感到安心——"不管我怎么滑地图，随时可以回到自己的位置"。

### 与设计系统的关系

- 引用Token: text-secondary(idle) / primary(active)
- 线宽: 1.5px (20px图标标准)
- 端点/拐角: round
- 系列: 地图操作图标组(I-01, I-02)

---

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 简洁(4-5笔) | 20px小尺寸下必须极简保证辨识 |
| 几何 vs 有机 | 几何+微圆角 | 导航/定位属于精确操作 |
| 平面 vs 立体 | 平面线性 | 与全局icon风格统一 |
| 规整 vs 随意 | 规整对称 | 表达精确定位 |

### 色彩策略

| 属性 | 值 | Token | 说明 |
|------|------|-------|------|
| 主色 | #A8A8B3 | text-secondary | idle态融入界面 |
| 激活色 | #4F8CFF | primary | 按下/定位中高亮 |
| 渐变 | 否 | — | 小图标不适合渐变 |

### 线条特征

| 属性 | 值 | 理由 |
|------|------|------|
| 线宽 | 1.5px | 20px标准 |
| 端点 | round | 柔和 |
| 拐角 | round | 统一 |
| 闭合性 | 混合(外圈闭合+内部开放) | 层次感 |
| 细节密度 | 少(4笔) | 20px保辨识 |

### 与同系列的关系

- 系列: 地图操作图标组
- 统一: 线宽1.5px, round端点, 视觉重量平衡
- 区分: 定位=同心圆+箭头 vs 视图切换=网格/菱形

---

## 3. 构图方案

### 参考框与安全区

```
参考框: 20×20 px
安全区: 四周2px
实际绘图区域: 16×16 px
视觉重心: 居中
```

### 图形描述

```
构图分层:

第1层(底层): 外圆环
  - 形状: circle stroke
  - 位置: 居中
  - 半径: 7px (直径14px)
  - 描边: 1.5px, text-secondary

第2层(顶层): 中心圆点
  - 形状: circle fill
  - 位置: 居中
  - 半径: 2.5px (直径5px)
  - 填充: text-secondary

附加(激活态): 四向短线
  - 上/下/左/右各一条2px短线
  - 从外圆环边缘向外延伸
  - 暗示"定位脉冲"
```

### 正负空间

- 正空间: ~30%
- 负空间作用: 让同心圆结构清晰可辨
- 关键留白: 外圆环与中心点之间

---

## 4. 变体设计

| 变体名 | 场景 | 与base差异 |
|--------|------|-----------|
| inactive | 默认idle态 | 纯stroke, text-secondary色 |
| active | 按下/定位中 | 色变primary, 中心点脉冲放大 |

### inactive → active 过渡

- 过渡属性: color(stroke+fill)
- 时长: 150ms
- 缓动: ease-default

---

## 5. 应用效果

### 最终视觉

- 位置: 地图右侧, FAB按钮上方, 固定定位
- 容器: 36×36px白底(Layer2)圆形按钮
- 与FAB间距: 12px
- 暗色背景上的浅灰线条icon，清晰可辨

### 应用后附加样式

```css
/* icon容器 */
width: 36px;
height: 36px;
border-radius: 999px;
background: #1C1C2E;
box-shadow: 0 2px 8px rgba(0,0,0,0.4);
display: flex;
align-items: center;
justify-content: center;
```

### 应用技术方案

| 步骤 | 操作 |
|------|------|
| 1 | 在容器内创建20×20 icon div |
| 2 | material-painter绘制(圆环+中心点) |
| 3 | export_and_apply |
| 4 | 清理background属性 |

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 20×20 px |
| 背景 | transparent |
| 目标节点 | LocateBtn 内的 20×20 icon div |

### 图形描述

```
一个同心圆定位图标：外圆环 + 中心实心圆点。

外圆环: 直径14px，居中于参考框，仅描边无填充
  - 描边色: #A8A8B3 (text-secondary)
  - 线宽: 1.5px

中心圆点: 直径5px，居中于参考框，实心填充
  - 填充色: #A8A8B3 (text-secondary)

整体传达"精确定位"的含义，视觉重量轻，作为辅助操作icon。
```

### 要求清单

- 色彩: 单色 #A8A8B3，引用 text-secondary Token
- 线宽: 1.5px，20px 图标标准线宽
- 风格: 线性描边(外圈) + 实心填充(中心点)
- 对齐: 完全居中于参考框
- 同系列一致性: 与 I-02 视觉重量平衡，统一线宽和端点

### 质量标准

- [ ] 圆环居中于参考框
- [ ] 1.5px 线宽在 20px 下清晰可辨
- [ ] 颜色与 text-secondary Token 一致
- [ ] 透明通道正确(无白底)
- [ ] 与 I-02 视觉重量平衡
