# I-05 空状态足迹插画 (empty-footprint)

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | I-05 |
| 名称 | empty-footprint |
| 类型 | Illustration |
| 参考框尺寸 | 120×120 px |

### 需求来源

| 来源 | 页面/组件 | 章节 | 需求描述 |
|------|---------|------|---------|
| 页面 | 01-home-map | visual.md 5 | "空状态'这里还没人留下足迹'引导情绪" |

### 要解决的问题

- 情感传达: 把「空无一物」的失落转化为「等你来创造」的激励
- 空白填充: 避免页面空洞感
- 行动引导: 配合CTA「发布第一条」

### 目标用户感受

> "这里还是空白画布——我可以成为第一个在这里留下印记的人！"

### 与设计系统的关系

- 引用Token: primary(主体色) + secondary(点缀) 低饱和度版本
- 遵循: 线性风格但更自由（插画允许稍微打破strict规则）
- 透明度: 整体60-80%，不抢CTA按钮注意力

---

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 中等(6-8个元素) | 空状态需要足够视觉填充 |
| 几何 vs 有机 | 几何为主+有机点缀 | 定位/路径=几何, 情感=有机曲线 |
| 平面 vs 立体 | 平面 | 与icon系统统一 |

### 色彩策略

| 属性 | 值 | Token | 说明 |
|------|------|-------|------|
| 主体 | rgba(79,140,255,0.3) | primary at 30% | 低饱和暗示而非抢夺 |
| 点缀 | rgba(124,92,252,0.2) | secondary at 20% | 增加层次 |
| 高光 | rgba(79,140,255,0.6) | primary at 60% | 定位pin的强调 |

## 3. 构图方案

### 参考框与安全区

```
参考框: 120×120 px
安全区: 8px
绘图区: 104×104 px
视觉重心: 居中偏上
```

### 图形描述

```
构图:
第1层(底): 虚线圆环(大, r≈40px, 虚线描边primary 15%)
  - 暗示"范围/区域"

第2层(中): 弯曲路径(从左下到右上的曲线)
  - 虚线描边, secondary 25%
  - 暗示"尚未走过的路径"

第3层(上): 定位Pin(居中偏上)
  - 实色填充 primary 50%
  - 下方一个微弱脉冲环(扩散圆)
  - 暗示"等待被激活的位置标记"

第4层(点缀): 3-4个小光点
  - 散布在路径周围
  - 2-4px, primary/secondary 15%
  - 暗示"潜在的其他足迹位置"
```

---

## 4. 变体设计

仅1个变体(empty态固定展示)。无需状态切换。

---

## 5. 应用效果

### 最终视觉

- 位置: 地图中心区域(empty状态时显示)
- 下方配文字: "这里还没有人留下足迹" + CTA按钮
- 整体透明度偏低，不会让用户误以为是真实地图数据
- 与暗色地图底和谐，像一个"等待被点亮"的空间

### 应用技术方案

| 步骤 | 操作 |
|------|------|
| 1 | empty状态容器内创建120×120 div |
| 2 | material-painter绘制 |
| 3 | export_and_apply |
| 4 | 清理background |

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 120×120 px |
| 背景 | transparent |
| 目标节点 | EmptyState 内的 120×120 插画 div |

### 图形描述

```
一个鼓励探索的空状态插画，传达"这里等你来点亮"的感觉。

组成元素（由远及近）：
1. 虚线大圆环: 直径约80px，居中
   - 描边: primary色 15%透明度
   - 线型: 虚线(dash)
   - 表达"范围/区域"

2. 弯曲路径线: 贯穿画面的虚线曲线
   - 描边: secondary色 25%透明度
   - 线型: 虚线(dash)
   - 表达"还没走过的路径"

3. 定位Pin(视觉焦点): 水滴形，居中偏上
   - 填充: primary色 50%透明度
   - 内有一个白色小圆点

4. 脉冲椭圆: Pin底部的扁椭圆
   - 填充: primary色 10%透明度
   - 表达"定位波"

5. 散落小光点(3个): 不同位置的小圆
   - 填充: primary/secondary 12-15%透明度
   - 大小: 2-4px
   - 表达"等待被发现的足迹"

整体透明度柔和，不抢CTA按钮注意力。
氛围: "黎明前的平静"，鼓励用户成为第一个打破沉默的人。
```

### 要求清单

- 色彩: 全部来自 primary/secondary Token 的低透明度变体(10-50%)
- 风格: 柔和、通透、氛围感
- 对齐: Pin 居中偏上作为视觉焦点
- 层次: 背景虚线→中景路径→前景Pin
- 约束: 整体不能太实/太重，要保持空态的"轻"

### 质量标准

- [ ] 整体透明度柔和不抢 CTA
- [ ] Pin 居中偏上作为视觉焦点
- [ ] 色彩全部来自 primary/secondary Token 的透明度变体
- [ ] 透明通道正确
- [ ] 120×120 尺寸下各元素可辨识

## 插画类额外章节

### 情感与叙事

| 字段 | 值 |
|------|------|
| 情感传达 | 鼓励+好奇 — "这里等你来点亮" |
| 画面叙事 | 一个沉睡的定位标记，周围是未走过的路径和等待发现的光点 |
| 主体元素 | 定位Pin(中心) + 路径线 + 范围圈 + 星光点 |
| 人物处理 | 无人物(用Pin替代人的存在感) |

### 构图参考

> 一个安静等待的定位标记悬浮在虚线圈的中心，脚下延伸出一条还没有人走过的虚线路径。四周散落几颗微弱的光点——它们暗示着这里终将被其他人的足迹点亮。整体氛围是"黎明前的平静"，鼓励用户成为第一个打破沉默的人。
