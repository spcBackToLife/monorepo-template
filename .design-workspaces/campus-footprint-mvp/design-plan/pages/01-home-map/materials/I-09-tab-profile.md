### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 20×20 px |
| 背景 | transparent |
| 目标节点 | TabProfile 内的 20×20 icon div |

### 图形描述

```
一个人物头像图标，表达"我的/个人中心"。

组成：
1. 头部: 圆形，直径约6px，居中偏上
   - 仅描边，1.5px
2. 肩部: 半圆弧线，宽约12px，在头部下方
   - 仅描边，1.5px
   - 弧线开口朝上，底部与参考框底边齐平

颜色: #6B6B7B (inactive态)，1.5px 线宽，round端点。
经典的人物头肩轮廓，最通用的"我的"图标形态。
```

### 质量标准

- [ ] 头+肩比例协调
- [ ] 居中
- [ ] 与其他Tab图标平衡
- [ ] 透明通道正确
