# I-03 发布加号图标 (publish-plus)

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | I-03 |
| 名称 | publish-plus |
| 类型 | Icon |
| 参考框尺寸 | 24×24 px |

### 需求来源

| 来源 | 页面/组件 | 章节 | 需求描述 |
|------|---------|------|---------|
| 页面 | 01-home-map | visual.md 3.5 | "FAB中心的发布加号,品牌统一" |

### 要解决的问题

- 功能识别: 全App最核心的「创建内容」操作
- 品牌统一: 加号是社交App的通用符号，但要有自己的品牌厚度

### 目标用户感受

> "我要在这里留下自己的足迹！"——行动冲动

### 与设计系统的关系

- 引用Token: text-primary(#F2F2F7 白色,在渐变按钮上)
- 线宽: 2px (比标准粗,匹配56px FAB的视觉重量)
- 系列: 独立(FAB专用)

---

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 极简(2笔) | 十字最简洁也最通用 |
| 几何 vs 有机 | 几何 | 精确十字 |
| 平面 vs 立体 | 平面 | 统一 |

### 色彩策略

| 属性 | 值 | Token | 说明 |
|------|------|-------|------|
| 主色 | #F2F2F7 | text-primary | 白色在渐变蓝紫底上高对比 |
| 渐变 | 否 | — | 图标本身单色,底色已有渐变 |

### 线条特征

| 属性 | 值 | 理由 |
|------|------|------|
| 线宽 | 2px | 56px FAB需要更粗保持比例 |
| 端点 | round | 柔和友好 |
| 细节密度 | 极少(2笔十字) | 最高辨识度 |

---

## 3. 构图方案

### 参考框与安全区

```
参考框: 24×24 px
安全区: 4px
绘图区: 16×16 px
视觉重心: 绝对居中
```

### 图形描述

```
两条线交叉:
  - 横线: (4, 12) → (20, 12), stroke:2px, round端点
  - 竖线: (12, 4) → (12, 20), stroke:2px, round端点
  - 颜色: #F2F2F7
```

### 正负空间

- 正空间: ~15% (两条细线)
- 负空间: 大量留白，十字形态纯粹
- 视觉重心: 精确居中

---

## 4. 变体设计

| 变体名 | 场景 | 差异 |
|--------|------|------|
| default | 正常态 | 白色2px十字 |

仅1个变体。FAB的状态变化(hover/pressed)由容器CSS处理，icon本身不变。

---

## 5. 应用效果

### 最终视觉

- FAB(56px)中心的白色十字
- 在蓝紫渐变底色上极高对比
- 配合FAB的glow呼吸动效，整体形成"会发光的创建按钮"

### 应用技术方案

| 步骤 | 操作 |
|------|------|
| 1 | FAB容器内创建24×24 icon div |
| 2 | material-painter绘制十字 |
| 3 | export_and_apply |
| 4 | 清理background |

---

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 24×24 px |
| 背景 | transparent |
| 目标节点 | FabBtn 内的 24×24 icon div |

### 图形描述

```
一个简洁的十字加号，表达"创建/发布"操作。

横线: 从左到右穿过中心，长度16px，居中
竖线: 从上到下穿过中心，长度16px，居中
两线正交于参考框中心点。

- 颜色: #F2F2F7 (text-primary，白色)
- 线宽: 2px（比标准粗，匹配56px FAB的视觉重量）
- 端点: round（柔和友好）
- 安全区: 四周4px留白
```

### 要求清单

- 色彩: 白色 #F2F2F7，在渐变蓝紫按钮上保证高对比
- 线宽: 2px（FAB专用加粗）
- 风格: 极简线性，仅两笔
- 对齐: 绝对居中，十字正交
- 端点: round（柔和）

### 质量标准

- [ ] 十字精确居中于参考框
- [ ] 2px 在 24px 框内视觉重量适当
- [ ] 白色在透明底上（无白底问题）
- [ ] round 端点柔和
- [ ] 横线和竖线等长(16px)
