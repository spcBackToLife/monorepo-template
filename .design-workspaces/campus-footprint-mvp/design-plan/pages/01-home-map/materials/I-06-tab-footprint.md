# I-06 足迹Tab图标 (tab-footprint)

---

## 1. 设计意图

| 字段 | 值 |
|------|------|
| 编号 | I-06 |
| 名称 | tab-footprint |
| 类型 | Icon |
| 参考框尺寸 | 20×20 px |

### 需求来源

| 来源 | 组件 | 需求描述 |
|------|------|---------|
| 页面 | 01-home-map | TabBar"足迹"Tab的图标，表达"位置足迹/脚印" |

### 要解决的问题

- 功能识别: 一眼认出"这是足迹/地图Tab"
- 状态反馈: 激活(primary蓝) vs 非激活(tertiary灰)

### 与设计系统的关系

- 系列: Tab图标组(I-06/I-07-tab/I-08-tab/I-09-tab) 统一风格
- 线宽: 1.5px，round端点
- 双色: active=#4F8CFF, inactive=#6B6B7B

---

## 2. 风格分析

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 | 3-4笔 | 20px Tab图标必须极简 |
| 几何 | 定位Pin简化形态 | 与App核心功能"位置"关联 |

### 色彩策略

| 属性 | 值 | Token |
|------|------|-------|
| 激活色 | #4F8CFF | primary |
| 非激活色 | #6B6B7B | text-tertiary |

---

## 3. 构图方案

```
参考框: 20×20, 安全区: 2px

图形: 简化的定位Pin(水滴形轮廓)
  - 外轮廓: 水滴形描边，宽约12px，高约14px，居中
  - 内圆点: 直径4px，在水滴头部居中
  - 线性描边风格(与I-01统一)
```

---

## 4. 变体设计

| 变体 | 颜色 |
|------|------|
| active | #4F8CFF (primary) |
| inactive | #6B6B7B (text-tertiary) |

---

## 5. 应用效果

- TabBar内，"足迹"文字上方
- 20×20，与文字间距2px

---

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 20×20 px |
| 背景 | transparent |
| 目标节点 | TabFootprint 内的 20×20 icon div |

### 图形描述

```
一个简化的定位 Pin 轮廓(水滴形)，表达"足迹/位置"。

组成：
1. 水滴形外轮廓: 上圆下尖，宽约12px高约14px，居中
   - 仅描边，无填充
   - 描边: #4F8CFF (active态), 1.5px

2. 内圆点: 直径4px，在水滴圆形部分居中
   - 实心填充: #4F8CFF

简洁线性风格，与导航图标组统一。
```

### 质量标准

- [ ] 水滴形对称
- [ ] 1.5px 线宽清晰
- [ ] 与其他3个Tab图标视觉重量平衡
- [ ] 透明通道正确
