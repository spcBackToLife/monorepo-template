# I-02 视图切换图标 (view-toggle)

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | I-02 |
| 名称 | view-toggle |
| 类型 | Icon |
| 参考框尺寸 | 20×20 px |

### 需求来源

| 来源 | 页面/组件 | 章节 | 需求描述 |
|------|---------|------|---------|
| 页面 | 01-home-map | visual.md 3.5 | "地图⟷列表视图切换识别" |

### 要解决的问题

- 功能识别: 用户知道点击可以切换地图/列表两种浏览方式
- 状态反馈: 当前是哪种视图一目了然

### 目标用户感受

> "我可以换一种方式浏览附近的动态"

### 与设计系统的关系

- 引用Token: text-secondary
- 线宽: 1.5px
- 系列: 地图操作图标组(I-01, I-02)

---

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 简洁 | 20px辨识度 |
| 几何 vs 有机 | 几何 | 网格/地图都是结构化形态 |
| 平面 vs 立体 | 平面线性 | 统一风格 |
| 规整 vs 随意 | 规整 | 工具类图标 |

### 色彩策略

| 属性 | 值 | Token | 说明 |
|------|------|-------|------|
| 主色 | #A8A8B3 | text-secondary | 工具图标默认色 |
| 渐变 | 否 | — | 小图标 |

### 线条特征

| 属性 | 值 | 理由 |
|------|------|------|
| 线宽 | 1.5px | 标准 |
| 端点 | round | 统一 |
| 细节密度 | 少(4-5笔) | 保辨识 |

---

## 3. 构图方案

### 参考框与安全区

```
参考框: 20×20 px
安全区: 2px
绘图区: 16×16 px
视觉重心: 居中
```

### 图形描述 — 列表态(当前为地图视图时显示)

```
4个小方块(2×2排列):
  - 每个方块: 6×6px, 圆角1px
  - 间距: 2px
  - 排列: 左上(3,3) 右上(11,3) 左下(3,11) 右下(11,11)
  - stroke: 1.5px, text-secondary
```

### 图形描述 — 地图态(当前为列表视图时显示)

```
简化地图符号:
  - 外框: 14×14px圆角矩形(2px圆角), stroke 1.5px
  - 内部: 一个小定位pin(简化菱形+圆点)居中偏上
```

---

## 4. 变体设计

| 变体名 | 场景 | 形态 |
|--------|------|------|
| show-list | 当前是地图,点击切列表 | 4方块网格 |
| show-map | 当前是列表,点击切地图 | 圆角框+pin |

### 变体间过渡

- 无动画过渡(瞬切)，因为整个视图会做交叉淡入淡出200ms
- 图标本身状态改变配合视图transition

---

## 5. 应用效果

### 最终视觉

- 位置: NavBar右侧按钮区
- 容器: 32×32px触摸热区
- 深色NavBar毛玻璃上的浅灰线条

### 应用技术方案

| 步骤 | 操作 |
|------|------|
| 1 | 创建20×20 icon div |
| 2 | material-painter绘制(列表态) |
| 3 | export_and_apply |
| 4 | 清理background |
| 5 | 用 visibleWhen 切换两种形态div |

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 20×20 px |
| 背景 | transparent |
| 目标节点 | ToggleBtn 内的 20×20 icon div |

### 图形描述

```
四个等大的圆角小方块，2×2 网格排列，表达"列表/网格视图"。

每个方块:
  - 尺寸: 6×6 px
  - 圆角: 1px
  - 仅描边，无填充
  - 描边色: #A8A8B3 (text-secondary)
  - 线宽: 1.5px

排列: 2×2 网格，方块间距 2px，整体居中于 20×20 参考框。
```

### 要求清单

- 色彩: 单色 #A8A8B3，引用 text-secondary Token
- 线宽: 1.5px
- 风格: 线性描边，微圆角
- 对齐: 4 方块完全对称，整体居中
- 同系列一致性: 与 I-01 视觉重量平衡

### 质量标准

- [ ] 4 方块对称分布
- [ ] 间距均匀(2px)
- [ ] 与 I-01 视觉重量平衡
- [ ] 透明通道正确(无白底)
