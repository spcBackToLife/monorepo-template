# 校园足迹社交 — 全局设计系统

> 风格定义：**午夜校园发光派对** — 暗色底色 + 活力荧光色
> 面向人群：18-24岁大学生，追求潮流、社交活跃、夜生活丰富

## 1. 色彩系统

### 1.1 主色推导

| 角色 | 色值 | HSL 分解 | 选择理由 |
|------|------|---------|--------|
| Primary (蓝) | #4F8CFF | hsl(220, 100%, 65%) | 科技感+信赖感，地图/位置的天然关联色，年轻人接受度最高的蓝 |
| Secondary (紫) | #7C5CFC | hsl(252, 97%, 68%) | 与Primary形成类似色和谐，增加神秘/社交感，「午夜」氛围核心色 |
| Accent (红) | #FF6B6B | hsl(0, 100%, 71%) | 高唤起度，用于点赞/心动/紧急，在暗色底上极度醒目 |
| Gold (金) | #FFB830 | hsl(40, 100%, 60%) | 稀缺/付费/VIP/定向动态，暗示价值与奖励 |

**主色关系分析**:
- Primary vs Secondary: 类似色（220° vs 252°）— 相邻色相保证和谐，32°色差足够区分
- Primary用途: 导航/按钮/链接/选中态/地图元素（面积≤15%）
- Secondary用途: 装饰/渐变过渡/捞人主题/次要CTA（面积≤8%）
- Accent用途: 点赞/心动/错误/紧急通知（面积≤5%）
- Gold用途: 付费/VIP/定向动态/特殊奖励（面积≤3%）

### 1.2 背景色层级

```
Layer 0 (最深) #0D0D14  hsl(240, 24%, 6%)   — App全局底色/地图底色
    ↑ (+3% lightness)
Layer 1       #141420  hsl(240, 23%, 10%)  — 卡片/列表项底色
    ↑ (+3% lightness)
Layer 2       #1C1C2E  hsl(240, 24%, 14%)  — 输入框/次级容器
    ↑ (+3% lightness)
Layer 3       #252540  hsl(240, 23%, 20%)  — 弹窗/Sheet底色
```

**选择理由**: 纯黑(#000)太沉闷，带紫蓝底色的极深灰（hue=240°）与Primary/Secondary色相统一，整体氛围更「午夜」而非「死寂」。

附加层:
- **毛玻璃层**: `rgba(13, 13, 20, 0.75)` + `blur(20px)` + `saturate(1.2)` + `border: 1px solid rgba(255,255,255,0.06)` — NavBar/Sheet/弹窗
- **遮罩层**: `rgba(0, 0, 0, 0.6)` — Modal/Sheet背景暗化

| 10% | 标签底色/选中态背景 | 明确可见但不抢内容 |
| 15% | 装饰光晕/角落氛围色 | 营造氛围但不遮挡 |
| 20% | 主装饰元素/强调背景 | 明确的视觉层次 |
| 40% | 已读/禁用态 | 明显弱化 |
| 75% | 毛玻璃底色 | 半遮挡+模糊 |

## 2. 间距系统

### 2.1 基数与 Scale

基数: 4px — 理由: 移动端最小可感知间距，4px网格确保像素对齐，8点系统的最小单位。

| Token | 值 | 典型使用场景 |
|-------|------|------------|
| space-1 | 4px | 图标与文字间距/标签内inline-padding |
| space-2 | 8px | 行内元素间距/气泡内小间距 |
| space-3 | 12px | 列表项内区块间距/卡片内小分组 |
| space-4 | 16px | 卡片padding/组间距/标准内间距 |
| space-5 | 20px | 页面水平安全距（左右padding） |
| space-6 | 24px | 大区块间分隔/Sheet内padding |
| space-8 | 32px | 页面级区域分隔 |
| space-10 | 40px | 页面顶部留白/大面积呼吸 |
| space-12 | 48px | 引导页/空状态的呼吸空间 |

### 2.2 按组件类型的间距规范

| 组件/场景 | 内间距(padding) | 元素间距(gap) | 外间距(margin) | 理由 |
|----------|:-:|:-:|:-:|------|
| 页面容器 | 0 20px | — | — | 左右安全距，内容不贴边 |
| 卡片 | 16px | — | 0 0 12px 0 | 卡片间呼吸感 |
| 列表项 | 12px 16px | 8px | 0 | 紧凑但可读 |
| 输入框 | 12px 16px | — | 0 0 16px 0 | 44px最小高度+舒适触摸 |
| 按钮(md) | 0 24px | 8px | — | 内容居中有呼吸 |
| 区块标题→内容 | — | 12px | 24px(上) | 标题与内容强关联 |
| Tab栏 | 0 | 0 | — | Tab项紧密 |
| Sheet/弹窗 | 20px 20px 24px | 16px | — | 比卡片更宽松 |
| 地图气泡 | 8px | 4px | — | 紧凑信息密度 |

## 6. 动效系统

### 6.1 缓动函数

| 名称 | 值 | 使用场景 | 视觉感受 |
|------|------|---------|---------|
| ease-default | `cubic-bezier(0.4, 0, 0.2, 1)` | 通用过渡/颜色变化/透明度 | 自然平衡 |
| ease-in | `cubic-bezier(0.4, 0, 1, 1)` | 退出/消失动画 | 加速离开 |
| ease-out | `cubic-bezier(0, 0, 0.2, 1)` | 进入/出现动画 | 减速到达 |
| spring | `cubic-bezier(0.175, 0.885, 0.32, 1.275)` | 弹跳/气泡出现/按钮释放 | 活力趣味 |

### 6.2 时长规范

| 时长 | 使用场景 | 为什么 |
|------|---------|--------|
| 100ms | 按钮press/release/触觉反馈 | 即时，不能有延迟感 |
| 150ms | hover态/微状态切换 | 可感知但不打断操作 |
| 200ms | Tab切换/简单过渡/颜色变化 | 平滑不拖沓 |
| 300ms | 页面转场/Sheet弹出/卡片展开 | 足够看清方向 |
| 400ms | 弹跳/强调动效/首次出现 | spring需要展示弹性 |
| 2000-3000ms | 循环动画(呼吸/脉冲/波纹) | 背景装饰节奏舒缓不焦躁 |

### 6.3 动画模式

| 模式 | 触发 | 效果 | 参数 |
|------|------|------|------|
| 按压反馈 | touch/click | scale(1→0.96→1) | 100ms ease |
| 进入弹跳 | 元素首次出现 | scale(0→1.05→1) + opacity(0→1) | 400ms spring |
| 脉冲呼吸 | idle循环 | scale(1→1.08→1) + glow扩散收缩 | 2.5s ease-in-out infinite |
| 波纹扩散 | 点击触发 | 圆形从触点扩散 | 600ms ease-out |
| 渐入上浮 | 列表项依次出现 | opacity(0→1) + translateY(8→0) | 200ms ease-out, stagger:50ms |
| 浮动 | 地图气泡idle | translateY(±2px) | 2.5s ease-in-out infinite |
| 旋转loading | 等待中 | rotate(0→360°) | 1s linear infinite |

## 7. 装饰配方（品牌视觉语言）

### 风格定义: 科技暗色 + 活力霓虹

```
主装饰: 渐变光晕
  - primary色(15-20%透明度) 大面积模糊圆
  - 位置: 角落/边缘，不占中心
  - blur: 60-80px，边缘极柔和

次装饰: 微光点/星芒
  - primary/secondary色(8-12%透明度) 小圆点
  - 3-5个散布在空白区域
  - 大小: 2-6px

微装饰: 噪点纹理
  - 全局叠加 opacity:0.02-0.03
  - 增加质感避免纯数字平面感

配色规则:
  - 装饰仅使用 primary/secondary 的低透明度变体
  - 不引入新色相
  - 装饰永远比内容暗/淡
```

### 装饰用量上限（按页面类型）

| 页面类型 | 装饰数上限 | 理由 |
|---------|:-:|------|
| 地图/Feed(内容丰富) | 2-3 | 内容本身即视觉，装饰极克制 |
| 登录/注册(品牌表达) | 3-5 | 表达品牌第一印象但焦点在表单 |
| 捞人/特殊玩法 | 5-8 | 游戏化场景允许更多视觉刺激 |
| 空状态/错误 | 5-8 | 需要填充空白+引导情绪 |
| 设置/列表(纯功能) | 0-1 | 装饰=噪音 |

### 图标风格规范

| 维度 | 规范 | 理由 |
|------|------|------|
| 风格 | 线性(stroke) | 轻量+现代+与暗色底形成线条美感 |
| 线宽 | 1.5px (20px图标) / 2px (24px图标) | 确保可辨识度 |
| 端点 | round | 柔和友好，匹配圆角系统 |
| 拐角 | round (4px内圆角) | 统一柔和感 |
| 激活态 | 填充(fill) + 渐变色 | 强烈区分选中/未选中 |
| 颜色 | 继承容器文字色/primary色 | 与上下文一致 |

## 8. 全局组件通用规范

### 按钮层级

| 层级 | 变体 | 使用场景 | 视觉强度 |
|------|------|---------|---------|
| L1 最强 | primary(渐变+glow) | 页面主操作（每页最多1个） | ★★★★★ |
| L2 次强 | gold(金色渐变+glow) | 付费/VIP操作 | ★★★★ |
| L3 中等 | secondary(半透明底) | 次要操作/确认 | ★★★ |
| L4 轻量 | ghost(透明底+边框) | 取消/辅助/可选操作 | ★★ |
| L5 最轻 | text(纯文字) | 链接/三级操作 | ★ |

### 状态统一规范

所有可交互元素遵循:
- **hover**: 亮度+5% 或 opacity+0.1
- **pressed**: scale(0.96-0.97) + 阴影收缩/消失
- **disabled**: opacity(0.4-0.5) + cursor:not-allowed
- **focused**: 2px outline(primary at 50%) + offset 2px
- **loading**: 内容替换为spinner + 禁用交互
