# M11 设置与隐私

> **优先级**：P1 - 隐私是位置社交产品的红线
> **决策依据**：D6 治愈风、隐私优先（位置数据敏感）
> **关联模块**：M5/M2/M6/M8/M10/M9 全部相关

---

## Step A: 用户故事

### 核心故事
- 作为**用户**，我希望一处集中管理所有偏好（账号/隐私/通知/外观/缓存）
- 作为**用户**，我希望对位置敏感信息有强控制力（"谁能看我位置"、"谁能捞到我"）
- 作为**用户**，我希望管理已登录设备，发现异常立刻下线
- 作为**用户**，我希望随时注销账号，且有清晰流程

### 扩展故事
- 作为**用户**，我希望一键开启/关闭"隐私模式"（不被捞、动态不进广场、足迹不可见）
- 作为**用户**，我希望管理拉黑列表
- 作为**用户**，我希望查看已读隐私协议、用户协议、社区公约
- 作为**用户**，我希望联系客服、提交反馈、举报问题

### 异常故事
- 作为**用户**，找回账号时遇到困难 → 客服人工通道兜底
- 作为**用户**，注销过程中改主意 → 缓冲期内可撤回（M5 已定义）

## Step B: 核心流程

### B1. 设置主菜单结构

```

├── 账号与安全（M5 联动）
│   ├── 手机号管理
│   ├── 修改密码

│   ├── 设备管理（已登录设备 + 强制下线）
│   ├── 注销账号
│
├── 隐私设置

│   ├── 公开动态是否进入跨校广场
│   ├── 主页字段可见性（性别/院系/年级/真实姓名）

│   ├── 主页访问记录开关
```

### B2. 隐私模式（一键开关）

```
开启"隐私模式"后：
- 不被捞到（M2 fishing_visibility = hidden）
- 动态不进跨校广场（plaza_share = false）
- 路过统计不计入（不上报 visit）
- 主页对陌生人完全隐藏
- 关闭已读
- 关闭主页访问记录
- 关闭"附近的好友"等所有发现机制（V1.x 若有）

退出隐私模式：恢复用户自定义偏好（不强制重置为默认）
```

### B3. 异常分支

```
├── 修改密码失败（旧密码错）   → 提示 + 找回入口
├── 设备强制下线生效失败（缓存）→ 重试机制
├── 注销过程账号被冻结        → 提示进入"封禁状态"，注销暂停
└── 缓存清理时网络中断        → 仅清理本地，云端不动
```

## Step C: 业务规则与边界 Case

### C1. 隐私默认值（出厂偏隐私）

| 设置 | 默认 |
|------|------|
| 谁能捞到我 | 仅同校 |
| 公开动态进广场 | 开（用户发布时可单独关） |
| 性别可见 | 公开 |
| 院系可见 | 仅好友 |
| 年级可见 | 仅好友 |
| 真实姓名可见 | 仅自己（永远） |
| 陌生人私信 | 关（仅好友/招呼通过临时会话可私信） |
| 已读 | 开 |
| 主页访问记录 | 关（默认不显示谁访问过我） |
| 通知（互动类） | 开 |
| 通知（运营类） | 开 |
| 免打扰时段 | 23:00 - 07:00 |

### C2. 业务规则

#### 安全敏感操作的二次校验
- 修改密码：旧密码 + 验证码
- 修改手机号：原手机验证码 + 新手机验证码 + （高风险时）人脸核身
- 注销账号：手机验证码 + 警告确认
- 设备强制下线：本地账号操作允许；远程操作需密码二次确认

#### 拉黑管理
- 拉黑列表展示对方头像 + 拉黑时间 + 解除按钮
- 解除拉黑后 24h 不可再加好友（防误操作）

#### 缓存清理
- 仅清理本地媒体缓存 + 数据库缓存
- 不影响登录态、已上传的内容
- 显示缓存大小 + 清理后大小

### C3. 边界 Case

| 场景 | 处理 |
|------|------|
| 用户开启隐私模式后立即又关闭 | 状态恢复其个人自定义偏好（保留自定义） |
| 多端设置不同步 | 偏好以服务端为准，本地仅缓存 |
| 通知设置改了但没下发 | 客户端轮询同步偏好 + 服务端事件触发更新 |
| 主题切换不生效 | 强制重启 App / 软重启 |

## Step E: 交互要点

- **设置项分组**：每个大类一个浅色背景卡片，组内 list 风
- **开关**：iOS 风格 toggle + 主题色
- **隐私模式开关**：单独大开关在隐私页顶部，开启时全屏 sheet 展示"将要做什么"清单
- **设备管理**：每个设备一行，"当前设备"标签 + 异常设备红色警示
- **客服入口**：常见问题 + 在线客服 + 提交工单
- **关于页**：版本号 + 协议链接 + Q版图标插画

## 风险标记

| 风险 | 缓解 |
|------|------|
| 隐私偏好太多用户找不到 | 提供"快速隐私模式" + 搜索框 |
| 默认值设过严影响活跃度 | 上线前 A/B 测试默认值的影响 |
| 注销流程被滥用刷数据 | 限频 + 缓冲期 + 资产校验 |
