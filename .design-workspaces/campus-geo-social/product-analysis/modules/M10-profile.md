## Step A: 用户故事

### 核心故事
- 作为**用户**，我希望有一个个人主页展示我的头像、昵称、简介、校园身份
- 作为**用户**，我希望主页是我的"足迹墙"——展示我发的动态、我曾路过的地点
- 作为**用户**，我希望从主页快速进入：我的胶囊、我的钱包、我的招呼、我的好友、设置
- 作为**用户**，我希望编辑资料，但实名信息（姓名/学号）不可改
- 作为**访客**，我希望看到他人主页时清晰知道与 TA 的关系（陌生/有好友请求中/好友/已拉黑）

### 扩展故事
- 作为**用户**，我希望我的"足迹地图"上看到所有我发过/路过的点位
- 作为**用户**，我希望统计数据有正反馈："你在 X 个地方留下印记"、"认识了 Y 位新朋友"
- 作为**用户**，我希望支持上传背景图（封面）增加个性化
- 作为**用户**，我希望访客记录隐私可控（默认不显示访问历史）

### 异常故事
- 作为**访客**，看陌生人主页时只能看公开信息（不暴露好友列表/钱包/胶囊）
- 作为**已被对方拉黑的访客** → 主页显示"该用户主页不可见"

## Step B: 核心流程

### B1. 自己的主页

```
入口：底部 Tab "我的"
布局（垂直滚动）：
1. 顶部封面 + 头像 + 昵称 + 简介
2. 数据卡片：动态数 / 路过次数 / 朋友数 / 胶囊数
3. 主校园 Badge + 切换主校园按钮（M5）
4. 功能宫格（4×N）：
   - 我的动态（M1）
   - 我的胶囊（M3）
   - 我的招呼（M2）
   - 我的好友（M6）
   - 我的钱包（M7）
   - 收藏的校园（M4）
   - 我的足迹（地图汇总）
   - 设置（M11）
5. 收藏的内容 / 收藏的卡片
6. 历史漫游过的校园
```

### B2. 他人的主页

```
入口：动态作者头像 / 评论区头像 / 招呼卡片 / 搜索结果
布局：
1. 顶部封面 + 头像 + 昵称 + 简介 + 院系/年级（如对方设置可见）
2. 关系状态徽章：好友 / 陌生人 / 已发好友请求 / 已拉黑
3. 主操作按钮（动态变化）：
   ├── 陌生人 → "加好友" + "私信"（在对方允许陌生私信时）
   ├── 已请求 → "请求中" 不可点
   ├── 好友 → "私聊" 主按钮 + 删除好友
   └── 已拉黑 → "解除拉黑"
4. TA 的公开动态 Feed
5. TA 的标志地点（最近常出现的几个点）
6. 共同好友（≥1 时展示）
7. 更多："举报" / "拉黑" / "复制 ID"
```

### B3. 编辑资料

```
设置 → 编辑资料：
- 头像（必有）
- 昵称（可改但 30 天 1 次）
- 一句话简介（≤ 50 字）
- 性别（可改）
- 年级（学信网每学年自动同步，可手动确认）
- 院系（同上）
- 兴趣标签（≤ 5 个）
- 封面图（可空，默认主题色背景）
不可改：
- 真实姓名 / 学号 / 身份证 / 学校（M5 强制）
```

### B4. 异常分支

```
├── 头像审核未通过      → 暂用默认头像 + 通知重传
├── 昵称冲突            → 推荐变体
├── 上传图片失败        → Toast 提示 + 重试
└── 主校园切换需重新认证→ 跳转 M5 流程
```

## Step C: 业务规则与边界 Case

### C1. 数据规则

| 字段 | 规则 |
|------|------|
| 头像 | 必有，jpg/png ≤ 5MB |
| 昵称 | 唯一 + 30 天可改 1 次 + 内容审核 |
| 简介 | ≤ 50 字 + 内容审核 |
| 封面 | 可空，jpg/png ≤ 10MB |
| 兴趣标签 | 系统预设 + 用户自填，每个 ≤ 8 字 |

### C2. 业务规则

#### 主页可见性
- 自己的主页：完全可见
- 好友：可见基础信息 + 公开动态 + 共同好友 + 朋友间标签（如"在 X 月日认识"）
- 陌生人：基础信息 + 仅公开动态 + 不显示好友列表 / 钱包 / 胶囊
- 已拉黑：完全不可见，显示"该用户主页不可见"

#### 隐私字段控制（与 M11 联动）
- 院系：默认对好友可见，对陌生人隐藏（用户可调）
- 年级：同上
- 性别：默认可见
- 真实姓名：默认对所有人不可见，用户可选"对好友可见"

#### 数据统计
- 动态数：实时
- 路过次数：实时（不暴露具体地点，仅总数）
- 朋友数：实时（陌生人看不到具体名单）
- 胶囊数：仅自己可见

### C3. 边界 Case

| 场景 | 处理 |
|------|------|
| 主校园切换是否触发重认证 | 切换需走 M5 学信网二次校验，确认对应学校学籍 |
| 注销账号期间访问主页 | 对外显示"该用户主页暂不可见" |
| 大量动态导致主页加载慢 | 分页 + 虚拟滚动 |
| 我的足迹地图数据隐私 | 仅自己可见，不暴露给好友 |

## Step E: 交互要点

### 自己的主页（D6 治愈风）
- **顶部封面区**：渐变奶油色背景 + 卡通插画装饰（樱花/落叶/泡泡）
- **数据卡片**：宽卡片，每个数字带柔和动画（数字滚动到位）
- **功能宫格**：圆角大方块，图标手绘风（不是线性图标）
- **足迹地图**：迷你地图聚合所有点位，金色光点表示已发动态/胶囊

### 他人主页
- **关系徽章**：右上角小标签（好友绿色/请求中黄色/拉黑灰色）
- **主操作 CTA**：底部固定按钮（私聊/加好友），始终可点
- **共同好友**：小圆头像气泡 + "你们的共同朋友：X 等 N 位"

### 编辑资料
- **每个字段独立行**：点击进编辑模式，回车保存
- **不可改字段**：标灰 + 锁图标 + tooltip
- **昵称改动剩余次数**：显示"本月还可改 0 次"

## 风险标记

| 风险 | 缓解 |
|------|------|
| 主页加载慢 | 数据预聚合 + 分块加载 |
| 隐私字段配置复杂 | 提供"快速隐私模式"一键设置（最严/中/最开放） |
| 用户头像被滥用（盗图） | 头像审核 + 报告通道 |
