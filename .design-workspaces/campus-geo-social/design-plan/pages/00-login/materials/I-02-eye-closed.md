# I-02 · eye-closed · 闭眼图标（密码密文态，默认）

> **页面**：00-login · 登录页
> **类型**：Icon（功能图标，P0 优先级）
> **绑定节点**：`pages/00-login/form-card/password-input`（条件：`state.view.passwordVisible !== true`，默认态）
> **来源**：`design-plan/pages/00-login/visual.md#§5-素材需求清单`

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | I-02 |
| 名称 | eye-closed |
| 类型 | Icon |
| 参考框尺寸 | 20×20 px（与 I-01 完全一致） |

### 需求来源

| 来源 | 页面/组件 | 章节 | 需求描述原文 |
|------|---------|------|------------|
| 页面 | 00-login | 素材清单 #I-02 | "切换密码为密文，常用控件" |

### 要解决的问题

- **功能识别**：告诉用户"现在密码以密文显示（默认安全），点我可以变为明文"
- **品牌强化**：与 I-01 形成"睁/闭"对应

### 目标用户感受

> 看到"闭眼"图标时，立刻明白"密码现在是隐藏的（安全）"。点击会变成"睁眼"。

### 与设计系统的关系

- **引用的 Token**：`$token:textSecondary`
- **遵循的规则**：iconSpec.outline / stroke 1.5px round
- **同系列**：与 I-01 共用外形参数

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 简洁（2 笔成型） | 闭眼比睁眼更简单 |
| 几何 vs 有机 | 偏几何（弧线 + 直线） | 标准化 |
| 平面 vs 立体 | 平面 | 全局一致 |
| 规整 vs 随意 | 规整 | 功能图标 |

### 色彩策略

| 属性 | 值 | Token 引用 |
|------|------|---------|
| 主色 | rgba(45,36,56,0.65) | `$token:textSecondary` |
| 辅色 | 无 | — |

### 线条特征

| 属性 | 值 | 理由 |
|------|------|------|
| 线宽 | 1.5px round | 与 I-01 一致 |
| 端点 | round | 全局一致 |
| 拐角 | round | 全局一致 |
| 闭合性 | 开放（两段弧线） | 强化"闭合"语义反差 |
| 细节密度 | 极少（2 笔） | 越简单越精致 |

### 与同系列的关系

- **与 I-01 必须共用外形参数**：外弧线 cx=10, cy=10，水平跨度同 14px（左右各 7px），保证切换时位置不跳
- I-02 是默认态（密码默认隐藏），所以视觉重量必须比 I-01 略轻或相等，不能更重

## 3. 构图方案

### 参考框与安全区

```
参考框: 20×20 px
安全区: 四周各 2px padding
实际绘图区域: 16×16 px
视觉重心: 居中（与 I-01 一致）
```

### 图形描述

```
构图分层:

第 1 层: 上眼睑弧线（闭合时看到的"半月"）
  - 形状: 二次贝塞尔弧线，开口朝下
  - 起点: (3, 10)
  - 终点: (17, 10)
  - 控制点: (10, 7)  ← 弧线向上拱起 3 px，形成"闭眼弯弯"形态
  - 填充: 无
  - 描边: textSecondary 1.5px round

第 2 层: 短斜线（暗示"睫毛/已关闭"）
  - 形状: 一条 1.5px 短斜线
  - 起点: (8, 13)
  - 终点: (12, 13.5)
  - 填充: 无
  - 描边: textSecondary 1.5px round

(可选优化：如果觉得过于简单，可以加 2 条短睫毛在弧线两端，但本次保持极简)
```

### 正负空间

- 正空间: ~ 12%（比 I-01 更少）
- 负空间作用: 闭合的眼睛"无内容"，留白即语义

### 视觉重量与平衡

- 视觉重心: 略偏上（弧线在上）
- 左右平衡: 对称（弧线和斜线都居中）

> 注意：虽然两者形态不同，但通过下方短斜线补足下半部分视觉重量，与 I-01 的瞳孔（实心圆）视觉重量大致相当，避免切换跳变。

## 6. 绘制要求

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框 | 20×20 px |
| 背景 | transparent |

### 图形描述（自然语言）

> 一只闭合的眼睛 outline 图标，由两条简单弧/线组成。  
> **主形（闭眼弯）**：从 (3, 10) 到 (17, 10)，中间向上拱起 3px 形成弯弯的"闭眼"弧线（控制点 (10, 7)），描边 textSecondary 1.5px round，无填充。  
> **下方短斜线（暗示已闭合）**：从 (8, 13) 到 (12, 13.5) 一条 1.5px 短斜线，textSecondary。  
>
> 视觉效果像一个微笑形的闭眼："~"。极简而温柔。

### 要求清单

- **色彩**：仅 `$token:textSecondary`
- **线宽**：1.5px round
- **风格**：开放线条（开放性反映"闭合"语义）
- **对齐**：水平居中
- **同系列一致性**：与 I-01 同位置同色同线宽，视觉重量近似

### 质量标准

- [ ] 在 20×20 内水平居中
- [ ] 弧线弧度自然（不像直线也不过弯）
- [ ] 切换 I-01 ↔ I-02 时无位置跳变
- [ ] 透明背景
