# I-02 · eye-closed · 密码隐藏图标

---

## 1. 设计意图

| 字段 | 值 |
|------|------|
| 编号 | I-02 |
| 名称 | eye-closed |
| 类型 | Icon |
| 参考框尺寸 | 20×20 px |

### 需求来源

| 来源 | 位置 | 需求 |
|------|------|------|
| 组件 | form-card/password-input 右侧 | 切换密码可见性（当前=隐藏） |

### 与设计系统的关系

- 同 I-01：outline 2px / round / textSecondary
- 与 I-01 构成一对（睁眼/闭眼）

---

## 2. 风格分析

与 I-01 完全一致（同族图标），唯一差异是造型变为"闭合+斜线"。

### 线条特征

| 属性 | 值 |
|------|------|
| 线宽 | 2px |
| 端点 | round |
| 拐角 | round |
| 细节密度 | 少(3-4笔: 眼轮廓+瞳孔+斜线) |

---

## 3. 构图方案

```
参考框: 20×20 px
安全区: 四周 3px
绘图区域: 14×14 px 居中
```

### 图形描述

```
第1层: 眼睛轮廓（同 I-01）
  - 杏仁形双弧线，14×8px

第2层: 瞳孔
  - 直径 4px 实心圆

第3层: 斜线（表示"关闭"）
  - 从左下到右上的对角线，贯穿整个眼形
  - 描边: textSecondary 2px round
  - 两端略超出眼形 1-2px
```

---

## 4. 变体设计

| 变体名 | 使用场景 | 差异 |
|--------|---------|------|
| base(唯一) | 密码隐藏时显示 | — |

---

## 5. 应用效果

- 位置: 同 I-01（password-input 右侧 44×44 触摸区内）
- 交互: 点击切换回 I-01

---

## 6. 绘制要求

| 属性 | 值 |
|------|------|
| 参考框 | 20×20 px |
| 背景 | transparent |

### 图形描述

"一只被斜线划过的眼睛。杏仁形眼框（同 I-01）+ 中心瞳孔 + 一条从左下到右上的对角线贯穿眼形，表示'不可见'。斜线两端略超出眼形边缘。"

### 要求清单

- 色彩: rgba(45,36,56,0.65) 单色
- 线宽: 2px（含斜线）
- 风格: outline
- 对齐: 居中
- 一致性: 与 I-01 视觉重量平衡（斜线不应使整体过重）
