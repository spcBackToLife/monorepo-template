# B-01 · brand-logo · 品牌 Logo

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | B-01 |
| 名称 | brand-logo |
| 类型 | Brand |
| 参考框尺寸 | 64×64 px |

### 需求来源

| 来源 | 页面/组件 | 需求描述 |
|------|---------|---------|
| 页面 | 00-login/top-area | 品牌标识，用户进入App的第一视觉焦点 |

### 要解决的问题

- **品牌识别**: 一眼认出这是"校园地理社交"App
- **情感传达**: 年轻、治愈、有温度
- **氛围营造**: 为登录页顶部区域建立品牌锚点

### 目标用户感受

> 看到 Logo 时感到亲切、年轻、有趣——像是一个校园里温暖的社交角落标识。

### 与设计系统的关系

- 引用Token: primary(#FF6F91), secondary(#6FE2A8)
- 遵循规则: 圆角风格(rounded)、有机柔和(organic/playful)
- 同系列: 独立品牌素材，无同系列

---

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 简洁偏中 | 64px 尺寸需快速识别，不宜过复杂 |
| 几何 vs 有机 | 有机为主 | 与"青春治愈"、hand-drawn 美学一致 |
| 平面 vs 立体 | 平面 | 扁平化风格，不需要拟物 |
| 规整 vs 随意 | 略随意 | playful 但不失专业 |

### 色彩策略

| 属性 | 值 | Token | 说明 |
|------|------|-------|------|
| 主色 | #FF6F91 | primary | 草莓粉=品牌主色 |
| 辅色 | #6FE2A8 | secondary | 薄荷绿点缀，增加活力 |
| 是否渐变 | 否 | — | 纯色 logo 更易识别 |
| 透明度 | 不透明 | — | 品牌 logo 需 100% 清晰 |

### 线条特征

| 属性 | 值 | 理由 |
|------|------|------|
| 线宽 | 2.5-3px | 64px 参考框中需粗壮可见 |
| 端点 | round | 柔和有机 |
| 拐角 | round | 与全局圆角体系呼应 |
| 细节密度 | 中(6-10笔) | 适配 64px 可辨认 |

## 3. 构图方案

### 参考框与安全区

```
参考框: 64×64 px
安全区: 四周各 6px padding
实际绘图区域: 52×52 px
视觉重心: 居中
```

### 图形描述

```
构图概念：一个抽象的"地标气泡"造型（代表地理）+ 内嵌两个相连的圆弧（代表社交连接）

第1层(主体): 水滴/气泡轮廓
  - 形状: 底部尖角圆润的地图标记外形
  - 位置: 居中
  - 尺寸: 约 40×48px (含尖角)
  - 填充: #FF6F91 (primary)
  - 描边: 无

第2层(内部图形): 两个相连弧线
  - 形状: 两段 120° 圆弧，首尾相接形成"连接"符号
  - 位置: 气泡上半部分居中
  - 尺寸: 约 20×16px
  - 填充: 无
  - 描边: #FFFFFF 3px round

第3层(装饰): 薄荷绿小圆点
  - 形状: 实心圆
  - 位置: 气泡右上角外沿
  - 尺寸: 直径 8px
  - 填充: #6FE2A8 (secondary)
```

### 正负空间

- 正空间占比: ~55%
- 负空间作用: 确保气泡造型清晰可读
- 关键留白: 气泡内部白色弧线周围需呼吸空间

## 6. 绘制要求

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 64×64 px |
| 背景 | transparent |
| 目标节点 | top-area/logo |

### 图形描述

"一个粉色的地图标记气泡造型（底部有圆润尖角），内部有两段白色圆弧线首尾相接形成'连接'符号。气泡右上角有一个薄荷绿小实心圆点作为装饰。整体传达'地理+社交'概念。"

### 要求清单

- 色彩: 主体 #FF6F91, 内部弧线 #FFFFFF, 装饰点 #6FE2A8
- 线宽: 内部弧线 3px（粗壮可见）
- 风格: 填充(主体) + 线性(内部)混合
- 对齐: 整体居中于 64×64 参考框
- 品牌感: 有机圆润，不要过于几何/机械

### 质量标准

- [ ] 在 64×64 参考框内居中
- [ ] 气泡造型清晰可辨
- [ ] 内部连接符号在该尺寸下可读
- [ ] 透明通道正确
- [ ] 色彩与 Token 一致
