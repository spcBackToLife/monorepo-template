## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | B-01 |
| 名称 | brand-logo |
| 类型 | Brand |
| 参考框尺寸 | 64×64 px |

### 需求来源

| 来源 | 页面/组件 | 章节 | 需求描述原文 |
|------|---------|------|------------|
| 页面 | 00-login | 素材清单 #B-01 | "品牌锚点，承担'这是 校园地理社交 App'的核心识别" |
| 视觉分析 | 00-login | §3.5-图标与图形 | "手绘 outline + 主色实色填充关键部位，64×64" |

未来同样素材会复用到 00-splash 启动页（128×128）、00-register 注册页（48×48）、11-about 关于页（96×96）。本素材是**品牌资产种子**。

### 要解决的问题

- **功能识别**：让用户在任何界面看到这个 Logo 就立刻知道"这是 校园地理社交 App"
- **品牌定位传达**：通过"地理坐标 + 校园建筑"的图形组合，直观说明产品的双核心价值（地理 × 校园）
- **氛围营造**：手绘 outline 风 + 主色填充让 Logo 不冷硬，符合"青春治愈"主题
- **情感传达**：粉色填充让 Logo "活泼可爱"，区别于市面冷蓝企业 App
- **品牌强化**：建立"草莓粉 + 圆角 outline"的品牌视觉符号

### 目标用户感受

> 看到这个 Logo 时，用户应该想到：**"哦，是那个能在校园里捞人、看到附近动态的好玩 App"**——既有产品功能联想，又有"被陪伴/有意思"的情感色彩。

不应让用户联想到：教务系统、严肃工具、企业服务。

### 与设计系统的关系

- **引用的 Token**：
  - 主线条色：`$token:textPrimary` rgba(45, 36, 56, 0.92)
  - 强调填充色：`$token:primary` #FF6F91
  - 辅助点缀色：`$token:secondary` #6FE2A8（可选叶子装饰）
- **遵循的规则**：
  - iconSpec.stroke.width = 2px round（与全局 outline 图标同族）
  - iconSpec.stroke.linecap = round
  - iconSpec.stroke.linejoin = round
- **同系列约束**：品牌素材族（B-xx）目前仅本个，未来 B-02（App 图标）/ B-03（splash 大 Logo 变体）需保持核心几何形态一致

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | **偏简洁**（3-4 笔成型） | 64×64 尺寸下细节多就糊；Logo 必须缩到 32×32 仍清晰 |
| 几何 vs 有机 | **几何为主，转角圆润** | 圆角矩形（建筑）+ 水滴（定位）都是几何，但用 round corner 化解冷硬 |
| 平面 vs 立体 | **平面**（无渐变、无阴影） | 与全局 outline 一致；立体 Logo 在小尺寸下细节丢失严重 |
| 规整 vs 随意 | **规整**（坐标精确） | 品牌素材必须可控可缩放；手绘感来自圆头线条而非随意笔触 |

### 色彩策略

| 属性 | 值 | Token 引用 | 说明 |
|------|------|---------|------|
| 主色（线条） | rgba(45, 36, 56, 0.92) | `$token:textPrimary` | 主体描边，承担"深色稳定感" |
| 辅色（填充） | #FF6F91 | `$token:primary` | 仅填充水滴内部小圆 + 建筑窗户，强化"品牌粉" |
| 是否渐变 | 否 | — | 与平面 outline 风格冲突 |
| 透明度 | 否 | — | 全实色，确保品牌锐利 |

### 线条特征

| 属性 | 值 | 理由 |
|------|------|------|
| 线宽 | **2.0px**（基准 64×64） | 与全局 iconSpec 一致；缩到 32 时变 1px 仍可见 |
| 端点 | round | 圆头柔和，符合治愈风 |
| 拐角 | round（半径 2-3px） | 内拐圆角避免尖锐 |
| 闭合性 | 主要闭合 + 局部开放 | 主形闭合稳重，细节小笔可开放（如建筑顶部装饰线） |
| 细节密度 | **少**（3-5 个主要图形元素） | 64×64 容量小；过密会糊 |

### 与同系列素材的关系

- 属于"品牌素材族"（B-xx），目前仅此一个
- 未来 B-02（应用图标 1024×1024）需保持同样的"水滴 + 建筑"复合形态，仅尺寸放大、可加细节装饰
- 与功能图标族（I-xx）的区分：Logo 含**主色实色填充**，普通图标全 outline

## 3. 构图方案

### 参考框与安全区

```
参考框: 64×64 px
安全区: 四周各 4px padding
实际绘图区域: 56×56 px
视觉重心: 偏上（geo 标记在上，建筑在下，整体重心 Y 约 38%）
```

### 图形描述

**整体构图**：一个倒立的水滴形地理定位标记，从屏幕顶部"插"进下方的圆角矩形（建筑）顶部。视觉上像一个标准的地图 pin 标定在校园建筑上。

```
构图分层:

第 1 层（底层 z-0）: 建筑主体
  - 形状: 圆角矩形（rounded rect）
  - 位置: 居中下半部，y = 28-58
  - 尺寸: 宽 36，高 30，圆角 6
  - 填充: 透明
  - 描边: textPrimary 2px round

第 2 层 (z-5): 建筑窗户（强化"建筑"语义）
  - 形状: 两个小圆角矩形（窗）
  - 位置: 建筑内左右对称
    - 左窗: x = 21, y = 36, w = 8, h = 6
    - 右窗: x = 35, y = 36, w = 8, h = 6
  - 圆角: 1
  - 填充: primary 实色
  - 描边: 无

第 3 层 (z-10): 地理定位水滴标记
  - 形状: 水滴形（path 上半圆 + 下尖角）
  - 位置: 建筑中上方，居中
    - 水滴中心: x = 32, y = 16
    - 水滴外径: 直径 20，下方延出尖角到 y = 28（与建筑顶部对接）
  - 填充: 透明
  - 描边: textPrimary 2px round

第 4 层 (z-15): 水滴内圆（品牌色锚点）
  - 形状: 实心圆
  - 位置: 水滴几何中心 x = 32, y = 14
  - 半径: 5
  - 填充: primary #FF6F91 实色
  - 描边: 无
```

### 正负空间

- 正空间（实体图形）占比: ~ 30%（适合 Logo，呼吸感强）
- 负空间作用: 顶部 8px 留白 + 水滴左右各 12px 留白 = 让 Logo "可缩放"，缩到 32 时还有视觉余地
- 关键留白位置: 四角 4×4 px 必须空（防止贴边失去 Logo 感）

### 视觉重量与平衡

- 视觉重心位置: 偏上 38%（水滴在上吸睛 + 建筑在下做基础）
- 左右平衡: **对称**（窗户左右对称，水滴居中）
- 对称的理由: 品牌 Logo 必须稳重，非对称容易显得"歪"

## 5. 应用效果

### 在目标位置的最终视觉

在 00-login 页面 top-area 区块的视觉效果：

```
[64px 顶部 padding]
       [B-01 64×64 Logo 居中]
       [16px gap]
       [可选 brand-slogan「校园地理社交」h5 16px 600 textPrimary 居中]
       [bounce-in 400ms spring 入场]
```

- **与周围元素的关系**：
  - 与 slogan 文字垂直 16px 间距
  - 与下方 mode-toggle 间距 48px（xl）
  - 左右居中对齐于页面横轴
- **在深色/浅色背景上的表现**：
  - 浅色奶油背景（#FFFAF6）：完美（设计基准）
  - 深色 dark mode（#1F1318）：textPrimary 自动反色为 rgba(255,250,246,0.92)，主色 primary 不变，依然清晰
- **缩放清晰度**：
  - 64×64：精细完整
  - 32×32：线条变 1px，仍可识别（建议）
  - 16×16：细节丢失，不推荐用本素材；未来用 B-02 简化版

### 应用后附加样式

```css
/* top-area/logo 节点上的样式 */
width: 64px;
height: 64px;
display: block;
margin: 0 auto;
animation: bounce-in 400ms cubic-bezier(0.34, 1.56, 0.64, 1) 100ms both;

/* 不需要 filter / opacity，Logo 自带完整视觉 */
```

### 应用技术方案

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 创建空 logo div（64×64） | `top-area/logo` 节点 |
| 2 | material-painter 在素材画布绘制 | 按构图方案 §3 执行 |
| 3 | export_and_apply | projectId + materialId + nodeId="logo" |
| 4 | 清理 backgroundImage 残留 | style/reset + style/update（仅保留 width/height/animation） |

## 6. 绘制要求（给 material-painter）

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框尺寸 | 64×64 px |
| 背景 | transparent |
| 目标节点 | `top-area/logo`（空 div 64×64） |

### 图形描述（自然语言）

> 一个"地图 pin 标记 + 校园建筑"的复合 Logo。
>
> **建筑部分（底层）**：64×64 画布中间偏下，绘制一个 36×30 的圆角矩形（圆角 6px），仅描边 textPrimary 2px round 圆头圆拐，无填充。位置：x 14-50, y 28-58。
>
> **建筑窗户**：在建筑内左右对称两个小窗，每个 8×6 px 圆角 1px，纯填充 primary 色 (#FF6F91)，无描边。位置：左窗 x 21-29 y 36-42，右窗 x 35-43 y 36-42。
>
> **地理标记（顶层）**：建筑上方画一个水滴形 pin（标准地图 pin 形状）。水滴上半部是直径 20px 的圆（中心 x=32, y=16），下半部是尖角延伸至 y=28（恰好与建筑顶部接触）。描边 textPrimary 2px round，无填充。
>
> **水滴内部圆点**：在水滴几何中心 (32, 14) 画一个半径 5 的实心圆，填充 primary 色，无描边——这是 Logo 的视觉锚点。
>
> 整体效果像：一个粉色心的地图标记钉在校园教学楼上。

### 要求清单

- **色彩**：仅 2 种色——主线条 `$token:textPrimary`（深紫棕）+ 主填充 `$token:primary`（草莓粉）
- **线宽**：2.0px round（端点+拐角都圆）
- **风格**：outline 为主 + 局部纯色填充作锚点
- **对齐**：所有元素严格居中（左右对称），整体居中于 64×64
- **同系列一致性**：与全局 iconSpec 一致（outline 2px round）

### 质量标准

- [ ] 在 64×64 参考框内居中，安全区 4px 完整
- [ ] 线宽 2px 在 64×64 下清晰，缩到 32×32 仍可识别
- [ ] 颜色严格使用 `$token:textPrimary` 和 `$token:primary`（不可手填 hex）
- [ ] 透明背景（无白底）
- [ ] 水滴下尖角与建筑顶部精确接触（无缝隙、无重叠 > 1px）
- [ ] 左右窗户位置对称（误差 ≤ 0.5px）
