# D-02 · pink-circle · 粉色圆形装饰

---

## 1. 设计意图

| 字段 | 值 |
|------|------|
| 编号 | D-02 |
| 名称 | pink-circle |
| 类型 | Decoration |
| 参考框尺寸 | 80×80 px |

### 需求来源

| 来源 | 位置 | 需求 |
|------|------|------|
| 页面 | 00-login 右上角 | 品牌色氛围装饰，打破矩形单调 |

### 要解决的问题

- **氛围营造**: 为页面增加"有机柔软"层次
- **品牌强化**: 草莓粉色签名
- **空间感**: 打破纯白方正的单调

### 与设计系统的关系

- 引用Token: primary `#FF6F91` at 15% opacity
- 遵循规则: decorationRules.background=solid, 装饰密度=moderate
- 同系列: D-02 + D-03 构成登录页装饰对（粉圆+薄荷叶，对角放置）

---

## 2. 风格分析

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 极简 | 单一几何形，作为氛围不抢内容 |
| 几何 vs 有机 | 有机几何 | 圆形但边缘可略模糊（不完全硬边） |
| 平面 vs 立体 | 平面 | 无立体感需求 |

### 色彩策略

| 属性 | 值 | Token | 说明 |
|------|------|-------|------|
| 主色 | #FF6F91 | primary | 品牌主色 |
| 透明度 | 15% | — | 氛围层不干扰内容 |
| 是否渐变 | 可选径向渐变 | — | 中心略浓→边缘淡出 |

---

## 3. 构图方案

```
参考框: 80×80 px
安全区: 无（装饰可溢出）
视觉重心: 居中
```

### 图形描述

```
单层: 模糊圆形
  - 形状: 正圆（ellipse）
  - 位置: 居中
  - 尺寸: 直径 70-80px
  - 填充: radial-gradient(circle, rgba(255,111,145,0.20) 0%, rgba(255,111,145,0.05) 70%, transparent 100%)
  - 描边: 无
  - 效果: 边缘柔化（模糊3-5px）
```

## 5. 应用效果

- 位置: 页面右上角，部分溢出视口 (约 40% 可见)
- z-index: 氛围层 (z-5)，在内容层之下
- 背景: 顶部粉色渐变之上
- 注意: 不能遮挡任何交互元素

### 应用后附加样式

```css
position: absolute;
top: -20px;
right: -20px;
width: 80px;
height: 80px;
opacity: 1; /* 透明度已内置到径向渐变中 */
pointer-events: none;
z-index: -1;
```

---

## 6. 绘制要求

| 属性 | 值 |
|------|------|
| 参考框 | 80×80 px |
| 背景 | transparent |

### 图形描述

"一个柔和的粉色圆形光斑。中心颜色较浓(opacity ~20%)，向边缘径向渐淡直至透明。整体像一团柔和的粉色光晕。"

### 要求清单

- 色彩: #FF6F91 为基色，通过径向渐变从 20% → 5% → 0% 透明度过渡
- 风格: 填充，无描边
- 对齐: 居中
- 边缘: 柔和过渡，不要硬边

### 装饰特殊规格

| 字段 | 值 | 理由 |
|------|------|------|
| 模糊度 | 3-5px (内置于渐变) | 柔和氛围感 |
| 混合模式 | normal | 在浅色底上正常混合即可 |
| 动画 | 无 | 静止装饰，不分散注意力 |
| z-index | -1 | 必须在内容层之下 |

### 与其他装饰的关系

- 角色: 主装饰（粉色=品牌色=更醒目）
- 视觉重量: 弱（低透明度）
- 与 D-03 关系: 对角放置（D-02 右上，D-03 左下），构成视觉平衡
