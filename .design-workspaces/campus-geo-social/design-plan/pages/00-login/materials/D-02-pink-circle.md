## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | D-02 |
| 名称 | pink-circle |
| 类型 | Decoration |
| 参考框尺寸 | 80×80 px |

### 需求来源

| 来源 | 页面/组件 | 章节 | 需求描述原文 |
|------|---------|------|------------|
| 页面 | 00-login | 装饰元素 D-02 | "左上角装饰圆，填充顶部空白，强化主色出现" |

### 要解决的问题

- **氛围营造**：登录页除了顶部光晕外，左上角作为视觉引导的延伸点
- **品牌色覆盖**：在按钮主色之外，让品牌色"散布"在页面上，增强品牌色覆盖率
- **避免空白**：68px topNavHeight 下方有大量留白（在 logo 之前），需要装饰填充

### 目标用户感受

> 用户进入登录页时，视线在被顶部光晕和 Logo 吸引的同时，余光感知到"左上角有粉色色块"——这传达"页面是有内容的、有设计感的"，不是冷冰冰的工具登录页。

### 与设计系统的关系

- **引用的 Token**：`$token:primary` (#FF6F91) at 8% 透明度
- **遵循的规则**：
  - 装饰策略 `decorationRules.background = solid`（不做大面积渐变）
  - decoration-elements-guide 类型 = "有机形"（非正圆，模拟手绘水彩感）
- **同系列约束**：与 D-03（薄荷叶子）同属"页面氛围装饰"族，两者位置和视觉重量平衡

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 极简（1 个有机形） | 装饰必须低存在感 |
| 几何 vs 有机 | **有机**（不对称圆） | 与全局"圆角圆润"主题一致；正圆显得机械 |
| 平面 vs 立体 | 平面 + 透明度 | 全局无立体装饰 |
| 规整 vs 随意 | 略随意（手绘水彩感） | 强化"治愈"氛围 |

### 色彩策略

| 属性 | 值 | Token 引用 | 说明 |
|------|------|---------|------|
| 主色 | #FF6F91 at 8% 透明度 | `$token:primary` at 0.08 | 极淡，仅作背景氛围 |
| 辅色 | 无 | — | 单色装饰 |
| 是否渐变 | 否（边缘可微羽化） | — | 实色块 |
| 透明度 | **8%**（rgba(255,111,145,0.08)） | — | 装饰用量上限 |

### 线条特征

不适用（无线条，纯填充）。

### 与同系列的关系

- **与 D-01 顶部光晕**：D-01 在顶部居中 + 黄色 18%；D-02 在左上 + 粉色 8%。两者通过位置和颜色互补，避免装饰扎堆
- **与 D-03 叶子**：D-02 在左上 + 粉色；D-03 在右下 + 绿色。对角分布，色彩呼应主辅色

## 3. 构图方案

### 参考框与安全区

```
参考框: 80×80 px
安全区: 无（装饰本身允许溢出参考框，因为会用 transform 移位）
视觉重心: 略偏右下（不对称感）
```

### 图形描述

```
构图分层:

第 1 层（唯一层）: 有机不对称圆
  - 形状: 类椭圆 path（手绘水彩形态）
  - 描述: 一个略微"歪"的圆，水平直径 ~ 75px，垂直直径 ~ 70px
         右下方略微凸出 5-8px（模拟水彩颜料聚集感）
         左上方略微内凹 3-5px
  - 中心位置: 大致在 (40, 40)
  - 填充: primary at 8% rgba(255, 111, 145, 0.08)
  - 描边: 无
  - 边缘: 可选 1-2px 羽化（feather）使边缘柔和（如不支持则保留硬边）
```

**手绘水彩感的实现细节**：
- 不要用正圆 `<circle r=37>`，而是 path 绘制带不规则曲率的封闭形
- 控制点可参考：12 点钟、3 点钟、6 点钟、9 点钟方向，每个方向的"半径"略微随机化 ±5%
- 简化版（如果工具不支持复杂 path）：用椭圆 75×70 略向右下平移即可

### 正负空间

- 正空间: ~ 80%（装饰本身占大部分参考框）
- 负空间作用: 四周留白让形状"漂浮"

### 视觉重量与平衡

- 视觉重心: 略偏右下 (45, 45)
- 不对称: 是。不对称是"手绘水彩"的核心特征

## 4. 变体设计

| 变体名 | 使用场景 | 与 base 差异 |
|--------|---------|-----------|
| base | 默认（00-login 左上） | — |
| breathing | 可选呼吸动效 | scale 1.0 ↔ 1.05 / 6s ease-in-out infinite |

> 本素材主要使用 base，breathing 变体仅在需要"活的氛围"时启用（如启动页过渡）。本登录页**用 base**。

### 变体详细规格

#### breathing

| 属性 | base 值 | breathing 值 | 变化理由 |
|------|--------|---------|---------|
| transform | none | scale(1) ↔ scale(1.05) | 缓慢"呼吸" |
| 动画 | 无 | `breath 6s ease-in-out infinite` | 周期足够慢避免视觉疲劳 |

### 变体间过渡

base → breathing 通过 CSS class 切换，不需要素材本体变化。

## 装饰特殊规格

| 字段 | 值 | 理由 |
|------|------|------|
| 模糊度 | 0（可选 2px feather） | 8% 透明度本身就柔和，不需要额外模糊 |
| 混合模式 | normal | 8% 透明度足够，不需要叠加模式 |
| 动画 | 无 | 登录页保持稳定，避免装饰干扰视线 |
| z-index | 0（最底层，pointer-events:none） | 装饰必须在内容层之下 |

### 与其他装饰的关系

- 本装饰在装饰体系中的角色：**微装饰**（视觉权重 1）
- 视觉重量：弱（8% 透明度，几乎看不到，仅在余光感知）
- 与 D-01 顶部光晕：D-01 是主装饰（权重 2），D-02 是辅；两者位置错开（D-01 居中顶部，D-02 偏左上）
- 与 D-03 薄荷叶子：对角分布（D-02 左上 + D-03 右下），主辅色呼应

## 6. 绘制要求

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框 | 80×80 px |
| 背景 | transparent |
| 目标节点 | `_page` root 内的 decor-pink-circle div |

### 图形描述（自然语言）

> 一个略微不对称的有机圆形色块（手绘水彩感）。  
> 中心约在 (40, 40)，整体水平直径约 75，垂直直径约 70；右下方略凸出 5-8px，左上方略内凹 3-5px，形成不完美的"手绘圆"。  
>
> 填充色 rgba(255, 111, 145, 0.08)——极淡的草莓粉，几乎透明，纯背景氛围作用。无描边。  
>
> 边缘可选 2px 羽化使其更"柔和"（如工具不支持则保留硬边，0.08 透明度本身就足够柔和）。  
>
> 视觉效果：在奶油白底色上若隐若现的粉色色块，像被太阳晒过的褪色印记。

### 要求清单

- **色彩**：仅 `$token:primary` at 8% 透明度
- **形状**：有机不对称圆（非正圆）
- **风格**：纯填充，无描边，无渐变
- **对齐**：基于 80×80 参考框，中心略偏右下
- **同系列一致性**：与 D-03 同属页面装饰族，但色彩、形状、位置都不同（互补）

### 质量标准

- [ ] 在 80×80 内位置正确
- [ ] 不对称感明显但不夸张
- [ ] 颜色严格 8% 透明
- [ ] 透明背景（无白底）
- [ ] 在奶油白背景上"若隐若现"（不能完全看不见，也不能太显眼）
