# I-01 · eye-open · 睁眼图标（密码明文态）

> **页面**：00-login · 登录页
> **类型**：Icon（功能图标，P0 优先级）
> **绑定节点**：`pages/00-login/form-card/password-input`（条件：`state.view.passwordVisible === true`）
> **来源**：`design-plan/pages/00-login/visual.md#§5-素材需求清单`

---

## 1. 设计意图

### 基本信息

| 字段 | 值 |
|------|------|
| 编号 | I-01 |
| 名称 | eye-open |
| 类型 | Icon |
| 参考框尺寸 | 20×20 px |

### 需求来源

| 来源 | 页面/组件 | 章节 | 需求描述原文 |
|------|---------|------|------------|
| 页面 | 00-login | 素材清单 #I-01 | "切换密码为明文，常用控件" |
| 视觉分析 | 00-login | §3.5 | "outline 2px round" |

### 要解决的问题

- **功能识别**：告诉用户"现在密码以明文显示，点我可以变为密文"
- **品牌强化**：与全局 outline 图标族保持一致

### 目标用户感受

> 看到"睁眼"图标时，立刻明白"密码现在能被看到"。点击会变成"闭眼"，符合"看到 ↔ 看不到"的心智模型。

### 与设计系统的关系

- **引用的 Token**：默认色 `$token:textSecondary` rgba(45, 36, 56, 0.65)
- **遵循的规则**：iconSpec.outline / stroke 2px round
- **同系列**：与 I-02 eye-closed 必须**完全同尺寸、同位置、同视觉重量**（仅形态变化）

## 2. 风格分析

### 设计方向

| 维度 | 选择 | 理由 |
|------|------|------|
| 简洁 vs 丰富 | 简洁（3 笔成型） | 20×20 小尺寸，多笔会糊 |
| 几何 vs 有机 | 偏几何（椭圆+圆） | 标准眼睛符号 |
| 平面 vs 立体 | 平面 | 与 outline 一致 |
| 规整 vs 随意 | 规整 | 功能图标 |

### 色彩策略

| 属性 | 值 | Token 引用 |
|------|------|---------|
| 主色 | rgba(45,36,56,0.65) | `$token:textSecondary` |
| 辅色 | 无 | — |
| 渐变 | 否 | — |
| 透明度 | 否（颜色本身已含 65% 透明度） | — |

### 线条特征

| 属性 | 值 | 理由 |
|------|------|------|
| 线宽 | 1.5px round | 20×20 小尺寸用 1.5 比 2 更精致 |
| 端点 | round | 全局一致 |
| 拐角 | round | 全局一致 |
| 闭合性 | 主形（外眼眶）闭合 + 内瞳孔实心圆 | 强化"睁眼"语义 |
| 细节密度 | 少（外眼眶 + 瞳孔 = 2 个元素） | 小图标必须极简 |

### 与同系列素材的关系

- 属于"通用功能图标族"（I-xx）
- **与 I-02 闭眼图标的关系**：
  - 外形必须严格同尺寸（眼眶椭圆参数一致）
  - 切换时视觉重量不能跳跃
  - 用户切换密码可见性时，眼睛 icon 应该看起来是"睁开 ↔ 闭合"的同一只眼

## 3. 构图方案

### 参考框与安全区

```
参考框: 20×20 px
安全区: 四周各 2px padding
实际绘图区域: 16×16 px
视觉重心: 居中
```

### 图形描述

```
构图分层:

第 1 层: 外眼眶（杏仁形/椭圆）
  - 形状: 水平椭圆
  - 位置: 居中 (cx=10, cy=10)
  - 尺寸: 水平半径 7, 垂直半径 4
  - 填充: 透明
  - 描边: textSecondary 1.5px round

第 2 层: 内瞳孔（实心圆）
  - 形状: 圆
  - 位置: 居中 (cx=10, cy=10)
  - 半径: 1.8
  - 填充: textSecondary 实色
  - 描边: 无
```

### 正负空间

- 正空间占比: ~ 25%
- 负空间作用: 给杏仁形眼眶呼吸感

### 视觉重量与平衡

- 视觉重心: 严格居中
- 左右平衡: 完全对称

## 6. 绘制要求

### 画布规格

| 属性 | 值 |
|------|------|
| 参考框 | 20×20 px |
| 背景 | transparent |
| 目标节点 | `form-card/password-input` 内嵌 icon div |

### 图形描述（自然语言）

> 一只睁开的眼睛 outline 图标。  
> 由两部分组成：  
> **外眼眶**：居中的水平椭圆（cx=10, cy=10，水平半径 7、垂直半径 4），仅描边 textSecondary 1.5px round，无填充。  
> **瞳孔**：居中实心圆（cx=10, cy=10, r=1.8），纯填充 textSecondary 实色。  
>
> 视觉效果像一只"惊讶"的圆眼睛，但因为尺寸小、颜色淡（65% 透明），整体显得克制不喧宾夺主。

### 要求清单

- **色彩**：仅 `$token:textSecondary`
- **线宽**：1.5px round
- **风格**：outline 主形 + 实心瞳孔点缀
- **对齐**：完全居中（cx=cy=10）
- **同系列一致性**：与 I-02 共用外眼眶参数

### 质量标准

- [ ] 在 20×20 内居中
- [ ] 1.5px 线宽清晰
- [ ] 颜色严格 textSecondary
- [ ] 透明背景
- [ ] 与 I-02 切换时无"跳变感"
