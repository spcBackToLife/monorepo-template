# D-03 · mint-leaf · 薄荷叶装饰

---

## 1. 设计意图

| 字段 | 值 |
|------|------|
| 编号 | D-03 |
| 名称 | mint-leaf |
| 类型 | Decoration |
| 参考框尺寸 | 60×40 px |

### 需求来源

| 来源 | 位置 | 需求 |
|------|------|------|
| 页面 | 00-login 左下角 | 辅色有机装饰，与 D-02 对角平衡 |

### 要解决的问题

- **有机感**: 叶形打破几何单调，强化"青春治愈"的自然感
- **辅色呈现**: 薄荷绿在页面中的存在位
- **视觉平衡**: 与右上角粉圆对角呼应

### 与设计系统的关系

- 引用Token: secondary `#6FE2A8` at 20% opacity
- 遵循规则: 有机形状(organic aesthetic)
- 同系列: D-02 + D-03 登录页装饰对

| 维度 | 选择 | 理由 |

## 3. 构图方案

```
参考框: 60×40 px
安全区: 无（装饰可溢出）
视觉重心: 中心偏左
```

### 图形描述

```
单层: 简化叶形
  - 形状: path 曲线，椭圆形叶子（一端尖，一端圆润）
  - 位置: 居中，略向左倾斜 15°
  - 尺寸: 约 50×30px
  - 填充: rgba(111, 226, 168, 0.20)
  - 描边: 无
  - 细节: 可选一条中脉线（同色 opacity 30%，1px）
```

## 6. 绘制要求

| 属性 | 值 |
|------|------|
| 参考框 | 60×40 px |
| 背景 | transparent |

### 图形描述

"一片简化的薄荷叶。椭圆形轮廓，左端圆润右端收尖，整体略呈弧线弯曲。内部可选一条轻微的中脉细线。颜色为低透明度薄荷绿，给人清新自然的感觉。"

### 要求清单

- 色彩: #6FE2A8 at 20% opacity
- 风格: 填充，无描边（或极细中脉线）
- 对齐: 居中于 60×40 框
- 有机感: 曲线自然流畅，不要过于几何/对称

### 装饰特殊规格

| 字段 | 值 | 理由 |
|------|------|------|
| 模糊度 | 0 (可选 2px) | 略比 D-02 清晰（叶形需可辨认） |
| 混合模式 | normal | 浅色底正常混合 |
| 动画 | 无 | 静止 |
| z-index | -1 | 在内容下方 |

### 与其他装饰的关系

- 角色: 次装饰（辅色=比主色弱）
- 视觉重量: 弱
- 与 D-02 关系: 对角平衡（左下 vs 右上），色彩互补（绿 vs 粉）
