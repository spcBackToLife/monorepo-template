# 设计计划总纲 - 登录页面 (00-login)

> 按照 design-planner-v3 技能重新设计，不受之前设计决策影响。
> 创建时间: 2026-06-06

## Phase A — 三层定位

### Layer 1 - 产品定位

**核心价值主张**（≤60 字）：
面向 18-22 在校生 + 23-26 应届生的校园社交 App 登录入口；首屏第一印象决定开学季 / 换机用户的留存与品牌心智。

**视觉差异化机会**（≥2 个真实竞品的视觉特征对照）：

| 产品 | 视觉特征 |
|---|---|
| 即刻 | 白底 + 黄黑高对比配色 / 圆角卡片 + 大字号标题 / 标签云 + 表情包文化 / 高密度信息流 + 强个性 tone of voice |
| 小红书 | 米白底 + 多色点缀 / 插画装饰 + 手写体标题 / 瀑布流双列 + 高饱和图片 / 女性化温度感 + 装饰繁多 |
| Notion/Linear | 纯白 + 纯灰阶 / Inter 等几何无衬线字体 / 网格化无装饰留白 / 工具感冷峻 + 极简但偏冷 |
| 微信/QQ | 浅绿 / 浅蓝品牌色 / 大头像 + 信息流密集 / 通用型容器 + 视觉退后让位关系链 / 无明显时代审美迭代 |

**用户视觉期待**（≤30 字）：
现代克制不土、圆角友好不冷、留白呼吸不堆功能。

**视觉差异化机会陈述**：
在校园社交品类里走'克制温度风'——比即刻更轻盈（不靠多色+表情包），比小红书更工具感（不靠插画+多图），比 Notion/Linear 更有温度（加暖底+蓝紫单色+节制几何装饰），差异化点 = 暖白底 + 蓝紫单一强调色 + 极细几何暗示性装饰。

---

### Layer 2 - 页面定位

**屏在用户旅程的位置**（≤50 字）：
App 启动入口屏（登录前唯一屏，决定第一印象后跳 01-home / 00-register / 00-forgot-password 三个出口）。

**屏对用户的核心收益**（≤30 字）：
5 步进主屏 + 留下'这设计不土'的第一印象。

**屏对产品的核心目标**（≤30 字）：
降低 18-22 焦虑首次流失 + 触达 23-26 审美心智。

**visualTiming 三档**（强制）：
- **0.5 秒**：BrandLogo 在屏顶 + 屏底偏暖,无视觉噪音,'这是个轻量入口屏'"
- **5 秒**：看清主操作 = 输入手机号 + 验证码登录,辅助操作 = 切换密码登录 / 注册 / 忘记密码
- **30 秒**：决策按下 SubmitBtn → 0.5s 见到 spinner → 进主屏

---

### Layer 3 - 用户场景

**用户进屏时的心理状态**（≤30 字）：
好奇 + 略防备 + 急切想跳过（首次焦虑 / 复登求快 / 找回焦躁三种心理叠加）。

**用户进屏时的紧迫度**：
medium

**用户离屏时的心理状态期望**（≤30 字）：
已加入校园圈层的预感 + 这家产品审美在线的好感。

**紧迫度对设计目标的影响**：
紧迫度 medium → 目标偏向 hierarchy / state-feedback；
视觉手段：清晰金字塔 + 多态视觉差异。

→ 提示 Phase B 提炼目标时可能含：mood-conveyance（降焦虑）+ cta-clarity（让用户快通过）+ trust-signal（合规但温和）+ state-feedback（锁定态温和）。

### 设计目标 5 字段（Step 3）

#### G1 (P0) — mood-conveyance

```json
{
  "id": "G1",
  "statement": "让用户进登录页 0.5 秒感受到'暖白米 + 克制温度'调性，区别于纯白冷峻 SaaS 风，降低首次注册防备",
  "whyMatters": "用户带'好奇 + 略防备'情绪进屏；视觉先安抚 = 注册转化前置条件；差异化定位（克制温度 vs 即刻多色 / 小红书插画 / Notion 冷峻）的可视化首层",
  "impactMode": "mood-conveyance",
  "successCriteria": [
    "屏底偏暖区（取屏中下 1/3 中心点像素采样）RGB 与 #FFFFFF 距离 ≥ 15pt（暖白米 + 装饰区融合达标，非纯白底）",
    "出现 ≥ 2 处 soft-glow 单一族装饰元素（圆形光斑 / 极细几何线，单一族不混杂），核心 alpha ≥ 20%（可分辨色相，非隐形）",
    "主色 #5B6CFF 与辅色 #A776FF 仅以'点缀'形态出现，不形成连续 > 30% 屏面积的实色块（避免冷峻 SaaS 调性）",
    "整屏灰阶占比 ≤ 35%（按截图主色统计；> 35% 即触发冷峻调）"
  ],
  "priority": "P0",
  "measureMethod": "Bash 调 scripts/screenshot-screen.mjs + Read 截图后，对屏底/装饰/主色块/灰阶分别取像素采样并按 4 条 SC 逐条核对"
}
```

**forbiddenSignals**:
- 屏底像素 #FFFFFF 或与之距离 ≤ 5pt
- 装饰核心 alpha < 15% 接近隐形
- 出现直角元素（border-radius < 4px）
- 主色 #5B6CFF 形成连续 > 30% 屏面积的实色块
- 灰阶占比 > 40%（含纯白底 + 灰字 + 灰边）
- 装饰族混杂（光斑 + 几何 + 插画 同时出现）

