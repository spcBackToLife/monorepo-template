# G1: mood-conveyance（P0）

> 让用户进登录页 0.5 秒感受到"克制温度"调性，区别于纯白冷峻 SaaS 风，降低首次注册防备

## 涉及元素

| 元素角色 | 节点 id | 当前问题 | 需要改什么 |
|---|---|---|---|
| 背景容器 | Root / BG | 纯白底，冷峻 | 加 soft-glow 暖色光斑装饰 |
| 装饰元素 1 | DecoGlow1（新增） | 不存在 | 画圆形光斑，径向渐变，alpha ≥ 20% |
| 装饰元素 2 | DecoGlow2（新增） | 不存在 | 画第二个光斑，对角配重，不混杂 |
| 主色点缀 | BrandLogo / SubmitBtn | 主色 #5B6CFF 可能形成大面积实色块 | 限制主色只以"点缀"形态出现 |

## 多维度改动

### 1. 颜色维度
- **背景**：不能是纯白 #FFFFFF，屏底偏暖区 RGB 与 #FFFFFF 距离 ≥ 15pt
- **主色**：#5B6CFF 与辅色 #A776FF 仅以"点缀"形态出现，不形成连续 > 30% 屏面积的实色块
- **灰阶占比**：整屏灰阶 ≤ 35%（按截图主色统计）

### 2. 装饰维度（soft-glow 系统）
- **读取策略**：必须先读 `decoration-systems/soft-glow.md`
- **装饰数量**：≥ 2 处 soft-glow 单一族装饰元素
- **装饰形态**：圆形光斑 / 极细几何线，单一族不混杂
- **核心 alpha**：≥ 20%（可分辨色相，非隐形）
- **配重策略**：对角配重（DecoGlow1 在左上，DecoGlow2 在右下）

### 3. 素材维度（material-painter）
- **BrandLogo**：需要真画素材（非占位虚线），用 material-painter 按 `soft-glow` 策略画
- **素材 brief**：指定装饰系统 `soft-glow`，参考 `decoration-systems/soft-glow.md`

## successCriteria 对账方法

1. "屏底偏暖区 RGB 与 #FFFFFF 距离 ≥ 15pt" → 截图，取屏中下 1/3 中心点像素采样
2. "出现 ≥ 2 处 soft-glow 装饰元素，核心 alpha ≥ 20%" → 截图，用取色器看光斑像素
3. "主色不形成连续 > 30% 屏面积实色块" → 截图，统计主色像素占比
4. "整屏灰阶占比 ≤ 35%" → 截图，转灰度后统计灰阶像素

## forbiddenSignals 检查清单

- [ ] 屏底像素 #FFFFFF 或与之距离 ≤ 5pt
- [ ] 装饰核心 alpha < 15%（接近隐形）
- [ ] 出现直角元素（border-radius < 4px）
- [ ] 主色 #5B6CFF 形成连续 > 30% 屏面积的实色块
- [ ] 灰阶占比 > 40%（含纯白底 + 灰字 + 灰边）
- [ ] 装饰族混杂（光斑 + 几何 + 插画 同时出现）

## ★ 沉淀到 schema 的结论

1. **新增装饰节点**：
   - `DecoGlow1`：div，position:absolute，宽高各 200px，径向渐变 `radial-gradient(circle, rgba(91,108,255,0.2) 0%, rgba(167,118,255,0.1) 50%, transparent 70%)`
   - `DecoGlow2`：div，position:absolute，宽高各 150px，径向渐变 `radial-gradient(circle, rgba(167,118,255,0.15) 0%, rgba(91,108,255,0.08) 60%, transparent 70%)`

2. **限制主色使用**：
   - SubmitBtn：主色填充，但宽度 100%（不是"连续 > 30% 屏面积"）
   - BrandLogo：主色只用于描边或强调点（不形成大面积实色块）

3. **写入 meta.design.visualStrategy**：
   ```json
   {
     "decorationSystem": "soft-glow(primary 20% + secondary 15%, 对角配重, 无混杂)",
     "colorStrategy": "60% 暖白底 + 30% 灰阶文字 + 10% 蓝紫点缀"
   }
   ```

4. **下一步**：调用 `material-painter` 画 BrandLogo 素材，必须先读 `decoration-systems/soft-glow.md`
