# G2: cta-clarity（P0）+ G3: trust-signal（P1）+ G4: brand-recognition（P0）+ G5: state-feedback（P1）

## G2: cta-clarity（P0）

> 让 SubmitBtn 成为首屏唯一主角，与 GetCodeBtn 形成 ≥ 4 权重差

### 涉及元素

| 元素角色 | 节点 id | 当前问题 | 需要改什么 |
|---|---|---|---|
| 主按钮 | SubmitBtn | 与 GetCodeBtn 视觉混淆 | 强化主色填充 + 字重 600 + 上方留白 24px |
| 次按钮 | GetCodeBtn | 与主按钮同色同字号 | 弱化为描边按钮 + 字号 14px + 字重 400 |

### 多维度改动

#### 1. 颜色维度
- **SubmitBtn**：主色 #5B6CFF 填充，白色文字，无边框
- **GetCodeBtn**：透明底，主色 #5B6CFF 描边 1px，主色文字，与 SubmitBtn 形成"填充 vs 描边"差异

#### 2. 尺寸维度
- **SubmitBtn**：height 48px，fontSize 16px，fontWeight 600，width 100%
- **GetCodeBtn**：height 36px，fontSize 14px，fontWeight 400，width auto

#### 3. 间距维度
- **SubmitBtn 上方留白**：marginTop 24px（与上方 FormItem 视觉断开）

#### 4. 视觉状态维度（5 态）
- **default**：主色填充，白色文字
- **hover**：主色填充，明度 + 10%
- **pressed**：主色填充，明度 - 10%
- **loading**：主色填充，白色 spinner（调 material-painter 画）
- **disabled**：灰色填充 #E5E7EB，白色文字，禁止点击

### ★ 沉淀到 schema 的结论

1. **SubmitBtn 样式**：
   ```json
   {
     "backgroundColor": "$token:colors.primary",
     "color": "#FFFFFF",
     "height": "48px",
     "fontSize": "16px",
     "fontWeight": "600",
     "width": "100%",
     "marginTop": "24px",
     "borderRadius": "12px"
   }
   ```

2. **GetCodeBtn 样式**：
   ```json
   {
     "backgroundColor": "transparent",
     "color": "$token:colors.primary",
     "height": "36px",
     "fontSize": "14px",
     "fontWeight": "400",
     "width": "auto",
     "border": "1px solid $token:colors.primary",
     "borderRadius": "8px"
   }
   ```

3. **SubmitBtn 5 态视觉状态**：
   - default：上述样式
   - hover：backgroundColor 明度 + 10%（用 CSS filter: brightness(1.1) 或 token）
   - pressed：backgroundColor 明度 - 10%（filter: brightness(0.9)）
   - loading：同 default + 左侧 spinner（materialProjectId）
   - disabled：backgroundColor `$token:colors.borderLight`，color `#FFFFFF`，cursor not-allowed

## G3: trust-signal（P1）

> 让协议勾选 + 错误反馈以友善温和方式触达，避免合规暴击

### 涉及元素

| 元素角色 | 节点 id | 当前问题 | 需要改什么 |
|---|---|---|---|
| 协议勾选 | PolicyCheckVisual（新增） | 用 native checkbox（黑色方框） | 自定义圆角勾选框 + 主色对勾 |
| 错误提示 | PhoneError / CredentialError | 可能用 #FF0000 红字暴击 | 用 theme.error #DD4747 + 12px 小字软化 |
| 协议链接 | TermsLink / PrivacyLink | 可能与正文同色无 underline | 主色 + underline 可视区分 |

### 多维度改动

#### 1. 协议勾选框（PolicyCheckVisual）
- **未勾态**：宽高 18×18px，border 1.5px solid `$token:colors.border`，borderRadius 4px，透明底
- **勾选态**：宽高 18×18px，backgroundColor `$token:colors.primary`，白色对勾（material-painter 画 SVG）
- **位置**：紧贴 PolicyText 左侧，间距 8px

#### 2. 错误提示（PhoneError / CredentialError）
- **文字色**：`$token:colors.error`（#DD4747，距纯红 #FF0000 ≥ 30pt）
- **字号**：12px（caption），非 14px（与正文同字号会抢戏）
- **位置**：紧贴对应 input 下方 4px
- **出现/消失**：≥ 200ms transition，非瞬间 pop

#### 3. 协议链接（TermsLink / PrivacyLink）
- **文字色**：`$token:colors.primary`，与正文 textPrimary 0.88 黑形成可视区分
- **textDecoration**：underline
- **hover**：color 明度 + 10%

### ★ 沉淀到 schema 的结论

1. **PolicyCheckVisual 节点**（新增，替代 native checkbox）：
   - type: "div"
   - props: { checked: boolean }
   - visualStates:
     - unchecked：border 1.5px solid `$token:colors.border`，透明底
     - checked：backgroundColor `$token:colors.primary`，白色对勾（materialProjectId）

2. **PhoneError / CredentialError 样式**：
   ```json
   {
     "color": "$token:colors.error",
     "fontSize": "12px",
     "marginTop": "4px",
     "transition": "opacity 200ms ease"
   }
   ```

3. **TermsLink / PrivacyLink 样式**：
   ```json
   {
     "color": "$token:colors.primary",
     "textDecoration": "underline",
     "fontSize": "12px"
   }
   ```

## G4: brand-recognition（P0）

> 让 BrandLogo 在屏顶撑起品牌识别第一锚点，承载"校园社交 + 现代时尚"双重定位

### 涉及元素

| 元素角色 | 节点 id | 当前问题 | 需要改什么 |
|---|---|---|---|
| 品牌 Logo | BrandLogo | materialProjectId 为空（占位虚线） | 调 material-painter 画真素材（非字母 C） |
| 品牌 Slogan | BrandSlogan | 可能与 Logo 间距过大 | 字号 16px + 与 BrandLogo 垂直间距 ≤ 12px |

### 多维度改动

#### 1. BrandLogo 素材（material-painter）
- **读取策略**：必须先读 `decoration-systems/soft-glow.md`
- **素材形态**：圆形/椭圆光斑 + 校园元素（书本/毕业帽/音符）+ 蓝紫渐变
- **素材尺寸**：参考框 200×200px，导出 PNG 100×100px @2x
- **色彩**：含 theme.primary #5B6CFF 或 secondary #A776FF（描边 / 主体形 / 强调点 / 渐变锚点）

#### 2. BrandSlogan 样式
- **字号**：16px（body-lg）
- **字重**：500（medium）
- **颜色**：`$token:colors.textPrimary` 0.88 黑
- **与 BrandLogo 间距**：≤ 12px（用 marginTop: -8px 或 gap: 8px）

### ★ 沉淀到 schema 的结论

1. **BrandLogo 节点**：
   - materialProjectId：非空（调 material-painter 画完后写入）
   - 宽高：100×100px（@viewport 393×852）
   - 位置：HeaderArea 中心，marginBottom 8px

2. **BrandSlogan 样式**：
   ```json
   {
     "fontSize": "16px",
     "fontWeight": "500",
     "color": "$token:colors.textPrimary",
     "opacity": "0.88",
     "marginTop": "8px"
   }
   ```

3. **下一步**：调用 `material-painter` 画 BrandLogo 素材，必须先读 `decoration-systems/soft-glow.md`

## G5: state-feedback（P1）

> 让 loginMode 切换 / 锁定态触发 / 倒计时变化等 5 种业务状态显著可分辨

### 涉及元素

| 元素角色 | 节点 id | 当前问题 | 需要改什么 |
|---|---|---|---|
| 登录模式 Tab | LoginModeTab（新增） | 可能无视觉差异 | active Tab 主色 + 底部 indicator，inactive Tab 灰色 |
| 锁定视图 | LockedView（新增） | 可能视觉冷峻像错误页 | 柔和视觉 + 锁形 icon（material-painter）+ 倒计时大字号 |
| 倒计时数字 | LockedCountdown | 可能字号小或非等宽 | 字号 ≥ 28px + monospace 等宽降级 |
| 加载 spinner | SubmitSpinner / GetCodeSpinner | 可能无 loading 态 | 调 material-painter 画 spinner SVG |

### 多维度改动

#### 1. LoginModeTab（code ↔ password 切换）
- **active Tab**：color `$token:colors.primary`，fontWeight 600，底部 indicator 3px 高主色
- **inactive Tab**：color `$token:colors.textSecondary`，fontWeight 400，无 indicator
- **切换动画**：≥ 200ms transition，indicator 位移动画

#### 2. LockedView（锁定态）
- **视觉**：柔和（非冷峻像 404），用 soft-glow 光斑背景
- **锁形 icon**：调 material-painter 画（柔和线条，非冷峻纯黑）
- **找回密码链接**：color `$token:colors.primary` + underline

#### 3. LockedCountdown（倒计时数字）
- **字号**：28px（display-sm）
- **字体**：monospace 等宽（避免数字变化 layout shift）
- **颜色**：`$token:colors.primary`

#### 4. SubmitSpinner / GetCodeSpinner（loading 态）
- **调 material-painter 画**：柔和 spinner SVG（蓝紫渐变，非纯灰）
- **尺寸**：16×16px（@viewport 393×852）

### ★ 沉淀到 schema 的结论

1. **LoginModeTab 样式**：
   - active：color `$token:colors.primary`，fontWeight 600，::after indicator 3px 高
   - inactive：color `$token:colors.textSecondary`，fontWeight 400

2. **LockedView 节点**（新增）：
   - 背景：soft-glow 光斑（radial-gradient，primary 10% alpha）
   - 锁形 icon：materialProjectId（material-painter 画）
   - LockedCountdown：fontSize 28px，fontFamily monospace

3. **SubmitSpinner / GetCodeSpinner**：
   - materialProjectId（material-painter 画 spinner SVG）
   - 尺寸：16×16px

4. **下一步**：调用 `material-painter` 画锁形 icon 和 spinner，必须先读 `decoration-systems/soft-glow.md`
