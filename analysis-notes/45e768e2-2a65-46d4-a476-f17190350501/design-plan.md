# 设计计划 — 校园社交产品

## 项目信息
- **项目 ID**: 45e768e2-2a65-46d4-a476-f17190350501
- **项目名称**: 校园社交产品
- **平台**: mobile (iPhone 15 Pro, 393×852)
- **主题**: 科技潮流风（暗色系 + 渐变 + 发光效果）

## 核心功能模块
1. **位置动态** - 基于位置发动态（视频、图文），可公开或定向
2. **捞人玩法** - 捞网方式捞别人的信息，付费玩法
3. **回忆功能** - 多年后走到这里能看到自己发的东西
4. **全球社交** - 基于位置锁定，查看某个校园的动态

## 关键决策记录
- **主题风格**: 科技潮流风（暗色系 #0A0E27，主色 #6C5CE7，辅助色 #00D2FF）
- **底部导航**: 4 个 tab（首页、捞人、消息、我的）
- **捞人玩法**: 参考"捞网"机制，付费增加趣味性
- **回忆功能**: 基于位置 + 时间轴，触景生情

## 技术要点
- 使用 `$token:colors.primary` 等主题变量
- 列表使用 `element.set_repeat`
- 条件显示使用 `element.set_visible_when`
- 表单使用 `element.set_bind`
- 页面跳转使用 `event.add_navigation`
- 素材绘制使用 `material-painter` 技能
